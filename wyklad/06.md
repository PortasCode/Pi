## 1. Kodowanie znaków

### Teoria

**Kodowanie znaków** to sposób przypisania liczb (kodów) konkretnym symbolom – literom, cyfrom, znakom specjalnym. Komputer nie rozumie liter, rozumie tylko liczby. Dlatego każdy znak musi mieć swój numer.

**Kod Morse'a** to historyczny przykład kodowania: każda litera to ciąg kropek (`.`) i kresek (`-`). Nie jest to kodowanie binarne w sensie komputerowym – tu informacja jest zakodowana w czasie (krótki sygnał = kropka, długi = kreska). Ważne: kody mają różną długość – litera `E` to jedna kropka, litera `Y` to cztery symbole (`-•--`). Jest to **kod prefiksowy** – żaden krótszy kod nie jest początkiem dłuższego.

**ASCII** (American Standard Code for Information Interchange) – standard z 1963 roku, używa 7 bitów, definiuje 128 znaków (0–127):
- 0–31 i 127: znaki sterujące (np. `LF` = nowa linia, `CR` = powrót karetki, `ESC` = escape)
- 32: spacja (`SP`)
- 33–47: znaki specjalne (`!`, `"`, `#`, `$`, ...)
- 48–57: cyfry `0`–`9` (kody 48–57, czyli `'0'` = 48)
- 65–90: wielkie litery `A`–`Z` (kod `'A'` = 65)
- 97–122: małe litery `a`–`z` (kod `'a'` = 97)

Kluczowa właściwość ASCII: litery i cyfry tworzą ciągłe zakresy. `'A'` + 1 = `'B'`, `'0'` + 5 = `'5'`. To jest wykorzystywane w programowaniu.

**EBCDIC** (Extended Binary Coded Decimal Interchange Code) – standard IBMowski z lat 60., używany w mainframe'ach. Używa 8 bitów (256 znaków), ale kolejność znaków jest inna niż w ASCII – litery i cyfry nie tworzą ciągłych zakresów. Nie jest kompatybilny z ASCII.

### Kod C

```c
#include <stdio.h>   /* biblioteka wejścia/wyjścia – printf, putchar */

int main(void) {
    /* ---- DEMONSTRACJA ASCII ---- */

    /* Wypisanie kodów ASCII kilku znaków */
    printf("Znak 'A' ma kod ASCII: %d\n", 'A');   /* wypisze: 65 */
    printf("Znak 'a' ma kod ASCII: %d\n", 'a');   /* wypisze: 97 */
    printf("Znak '0' ma kod ASCII: %d\n", '0');   /* wypisze: 48 */
    printf("Znak ' ' ma kod ASCII: %d\n", ' ');   /* wypisze: 32 */

    /* Różnica między 'a' a 'A' to zawsze 32 – to cecha ASCII */
    printf("Roznica miedzy 'a' a 'A': %d\n", 'a' - 'A');  /* wypisze: 32 */

    /* Cyfry w ASCII: '0'=48, '1'=49, ..., '9'=57 */
    printf("Cyfra '7' jako liczba: %d\n", '7' - '0');  /* wypisze: 7 */

    /* Wypisanie tabeli ASCII dla znaków drukowalnych (32–126) */
    printf("\n--- Znaki drukowalne ASCII (32-126) ---\n");
    for (int i = 32; i <= 126; i++) {     /* iterujemy po kodach */
        printf("DEC:%3d  HEX:%2X  ZNAK:'%c'\n", i, i, i);
        /* %d  = liczba dziesiętna            */
        /* %X  = liczba szesnastkowa (wielkie) */
        /* %c  = znak o danym kodzie           */
    }

    /* Znaki sterujące – nie wypisujemy ich jako znaków, bo mogą psuć terminal */
    printf("\n--- Wybrane znaki sterujące ASCII ---\n");
    printf("Kod  0 = NUL (null terminator)\n");
    printf("Kod  9 = HT  (poziomy tabulator)\n");
    printf("Kod 10 = LF  (nowa linia, \\n)\n");
    printf("Kod 13 = CR  (powrot karetki, \\r)\n");
    printf("Kod 27 = ESC (escape)\n");

    /* Sprawdzenie zakresu liter – ciągłość w ASCII */
    printf("\n--- Litery A-Z i ich kody ---\n");
    for (char c = 'A'; c <= 'Z'; c++) {   /* 'A'=65, 'Z'=90 – zakres ciągły */
        printf("%c=%d  ", c, (int)c);     /* rzutowanie na int żeby wypisać kod */
    }
    printf("\n");

    return 0;  /* program zakończony poprawnie – zwracamy 0 do systemu */
}
```

## 2. Unicode

### Teoria

**Unicode** to globalny standard przypisujący unikalny numer (tzw. **punkt kodowy**, ang. *code point*) każdemu znakowi używanemu na świecie. Cel: jeden standard dla wszystkich języków, symboli i emoji, zamiast dziesiątek niekompatybilnych standardów regionalnych.

Zakres punktów kodowych: **U+0000 do U+10FFFF** (szesnastkowo), co daje ponad 1 100 000 możliwych znaków. Aktualnie przypisanych jest około 150 000 znaków.

**Notacja:** punkty kodowe zapisuje się jako `U+` i cztery lub więcej cyfr szesnastkowych:
- `U+0000` = NUL (null)
- `U+0041` = `A` (wielka litera A)
- `U+0061` = `a` (mała litera a)
- `U+003D` = `=` (znak równości)
- `U+00F3` = `ó` (litera ó z polskiego alfabetu)
- `U+1F600` = 😀 (emoji uśmiechnięta twarz)

**Kompatybilność z ASCII:** punkty U+0000–U+007F są identyczne z ASCII. Dzięki temu zwykłe angielskie teksty działają bez zmian.

**Unicode to tylko lista przypisań** – nie określa jak znaki mają być zapisane w pamięci. Do tego służą kodowania UTF-8, UTF-16, UTF-32.

**Płaszczyzny Unicode:**
- Plane 0 (U+0000–U+FFFF): Basic Multilingual Plane (BMP) – większość języków świata
- Plane 1 (U+10000–U+1FFFF): Supplementary Multilingual Plane – np. emoji, historyczne pisma
- Plane 2–16: pozostałe, rzadziej używane

### Kod C

```c
#include <stdio.h>    /* printf */
#include <stdint.h>   /* typy uint32_t itp. */
#include <locale.h>   /* setlocale – ustawienia regionalne */

int main(void) {
    /* Ustawienie locale na UTF-8 – potrzebne do wypisywania znaków Unicode */
    /* Bez tego printf może nie wypisać poprawnie polskich liter             */
    setlocale(LC_ALL, "pl_PL.UTF-8");  /* ustawiamy polskie locale z UTF-8 */

    /* ---- PUNKTY KODOWE UNICODE ---- */
    /* Punkt kodowy to po prostu liczba – możemy go przechować w uint32_t   */

    uint32_t codepoint_A     = 0x0041;   /* U+0041 = 'A' */
    uint32_t codepoint_a     = 0x0061;   /* U+0061 = 'a' */
    uint32_t codepoint_o_ogon = 0x00F3;  /* U+00F3 = 'ó' (ó z kreską) */
    uint32_t codepoint_euro  = 0x20AC;   /* U+20AC = '€' (znak euro) */

    printf("Punkt kodowy A:    U+%04X (dec: %u)\n", codepoint_A, codepoint_A);
    /* %04X = szesnastkowo, min. 4 cyfry, dopełnione zerami */
    /* wypisze: U+0041 (dec: 65) */

    printf("Punkt kodowy a:    U+%04X (dec: %u)\n", codepoint_a, codepoint_a);
    /* wypisze: U+0061 (dec: 97) */

    printf("Punkt kodowy ó:    U+%04X (dec: %u)\n", codepoint_o_ogon, codepoint_o_ogon);
    /* wypisze: U+00F3 (dec: 243) */

    printf("Punkt kodowy euro: U+%04X (dec: %u)\n", codepoint_euro, codepoint_euro);
    /* wypisze: U+20AC (dec: 8364) */

    /* Kompatybilność z ASCII: pierwsze 128 punktów to dokładnie ASCII */
    printf("\nPierwsze 128 punktow Unicode = ASCII:\n");
    for (uint32_t cp = 0x20; cp <= 0x7E; cp++) {  /* od spacji do ~ */
        /* Znaki U+0000-U+007F mają te same wartości co ASCII */
        printf("U+%04X = '%c'\n", cp, (char)cp);  /* rzutowanie na char – bezpieczne w tym zakresie */
    }

    /* Polskie litery w Unicode – zakres Latin Extended */
    printf("\nPolskie litery i ich punkty kodowe:\n");
    /* Musimy użyć literałów UTF-8 zakodowanych wprost w source code */
    /* Kompilator potraktuje "ą" jako sekwencję bajtów UTF-8           */
    printf("a z ogonkiem (ą): U+0105\n");  /* U+0105 */
    printf("e z ogonkiem (ę): U+0119\n");  /* U+0119 */
    printf("o z kreska  (ó):  U+00F3\n");  /* U+00F3 */
    printf("s z kreska  (ś):  U+015B\n");  /* U+015B */
    printf("z z kreska  (ź):  U+017A\n");  /* U+017A */
    printf("z z kropka  (ż):  U+017C\n");  /* U+017C */
    printf("l z kreska  (ł):  U+0142\n");  /* U+0142 */
    printf("n z kreska  (ń):  U+0144\n");  /* U+0144 */
    printf("c z kreska  (ć):  U+0107\n");  /* U+0107 */
    printf("z z kreska  (ź):  U+017A\n");  /* U+017A */

    /* Zakres Unicode */
    printf("\nMaksymalny punkt kodowy Unicode: U+10FFFF = %u dziesiętnie\n",
           (unsigned int)0x10FFFF);
    /* wypisze: 1114111 */

    printf("Liczba mozliwych punktow kodowych: %u\n",
           (unsigned int)(0x10FFFF + 1));
    /* wypisze: 1114112 */

    return 0;
}
```

## 3. UTF-8

### Teoria

**UTF-8** (Unicode Transformation Format – 8-bit) to najważniejsze kodowanie Unicode. Używa od 1 do 4 bajtów na znak:

| Zakres punktów kodowych | Liczba bajtów | Wzorzec bitowy |
|-------------------------|---------------|----------------|
| U+0000 – U+007F | 1 bajt | `0xxxxxxx` |
| U+0080 – U+07FF | 2 bajty | `110xxxxx 10xxxxxx` |
| U+0800 – U+FFFF | 3 bajty | `1110xxxx 10xxxxxx 10xxxxxx` |
| U+10000 – U+10FFFF | 4 bajty | `11110xxx 10xxxxxx 10xxxxxx 10xxxxxx` |

**Jak to działa:**
- Znaki ASCII (U+0000–U+007F) mają pierwszy bit równy 0 – zakodowane identycznie jak w ASCII, jeden bajt. **To jest klucz do kompatybilności z ASCII.**
- Znaki wielobajtowe: pierwszy bajt zaczyna się od kilku jedynek (mówią ile bajtów łącznie), kolejne bajty zaczynają się od `10`.
- `x` w wzorcu to bity punktu kodowego.

**Zasada "tylko najkrótszy kod jest poprawny":** nie wolno zakodować znaku ASCII (np. `A` = U+0041) przy użyciu 2 bajtów, choć technicznie byłoby to możliwe. Takie "overlong sequences" są niepoprawne i mogą prowadzić do luk bezpieczeństwa.

**Przykład:** litera `ó` = U+00F3 = 0b11110011
- Mieści się w zakresie U+0080–U+07FF → 2 bajty
- Wzorzec: `110xxxxx 10xxxxxx`
- U+00F3 w binarnym: `000 11110 011`
  - 5 bitów do pierwszego bajtu: `00011` → `11000011` = 0xC3
  - 6 bitów do drugiego bajtu: `110011` → `10110011` = 0xB3
- Wynik: `0xC3 0xB3`

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strlen */
#include <stdint.h>   /* uint8_t, uint32_t */
#include <locale.h>   /* setlocale */

/* Funkcja wypisuje bajty stringa jako heksadecymalne */
void wypisz_bajty(const char *s) {
    /* Iterujemy po bajtach aż do '\0' */
    /* Rzutujemy na unsigned char żeby %02X dało właściwy wynik */
    for (size_t i = 0; s[i] != '\0'; i++) {          /* dla każdego bajtu */
        printf("%02X ", (unsigned char)s[i]);          /* hex, min. 2 cyfry */
    }
    printf("\n");
}

/* Funkcja koduje punkt kodowy Unicode na UTF-8 i wypisuje bajty */
/* Zwraca liczbę wynikowych bajtów */
int koduj_utf8(uint32_t codepoint, uint8_t *wynik) {
    if (codepoint <= 0x7F) {
        /* Zakres 1 bajtu: U+0000 - U+007F */
        wynik[0] = (uint8_t)codepoint;  /* jeden bajt, bezpośrednio wartość */
        return 1;                        /* zwracamy liczbę bajtów = 1 */
    } else if (codepoint <= 0x7FF) {
        /* Zakres 2 bajtów: U+0080 - U+07FF */
        /* Pierwszy bajt:  110xxxxx – wzorzec 0xC0, 5 bitów z codepoint */
        wynik[0] = (uint8_t)(0xC0 | (codepoint >> 6));
        /* Drugi bajt:     10xxxxxx – wzorzec 0x80, 6 bitów z codepoint */
        wynik[1] = (uint8_t)(0x80 | (codepoint & 0x3F));
        return 2;  /* dwa bajty */
    } else if (codepoint <= 0xFFFF) {
        /* Zakres 3 bajtów: U+0800 - U+FFFF */
        /* Pierwszy bajt:  1110xxxx – wzorzec 0xE0, 4 bity z codepoint */
        wynik[0] = (uint8_t)(0xE0 | (codepoint >> 12));
        /* Drugi bajt:     10xxxxxx – wzorzec 0x80, bity 11-6 z codepoint */
        wynik[1] = (uint8_t)(0x80 | ((codepoint >> 6) & 0x3F));
        /* Trzeci bajt:    10xxxxxx – wzorzec 0x80, bity 5-0 z codepoint */
        wynik[2] = (uint8_t)(0x80 | (codepoint & 0x3F));
        return 3;  /* trzy bajty */
    } else {
        /* Zakres 4 bajtów: U+10000 - U+10FFFF */
        /* Pierwszy bajt:  11110xxx – wzorzec 0xF0, 3 bity z codepoint */
        wynik[0] = (uint8_t)(0xF0 | (codepoint >> 18));
        wynik[1] = (uint8_t)(0x80 | ((codepoint >> 12) & 0x3F));
        wynik[2] = (uint8_t)(0x80 | ((codepoint >> 6) & 0x3F));
        wynik[3] = (uint8_t)(0x80 | (codepoint & 0x3F));
        return 4;  /* cztery bajty */
    }
}

int main(void) {
    setlocale(LC_ALL, "pl_PL.UTF-8");  /* locale z UTF-8 dla poprawnego printf */

    /* ---- Analiza bajtów UTF-8 w literałach ---- */
    printf("=== Bajty UTF-8 ===\n\n");

    /* Literał ASCII: każda litera = 1 bajt */
    const char *ascii = "ABC";
    printf("Tekst: \"%s\"\n", ascii);
    printf("strlen: %zu bajtow\n", strlen(ascii));   /* 3 bajty */
    printf("Bajty: ");
    wypisz_bajty(ascii);
    /* wypisze: 41 42 43 (hex dla A, B, C) */

    /* Literał z polską literą – wielobajtowy */
    const char *polish = "ołó";   /* o=U+006F(1B), ł=U+0142(2B), ó=U+00F3(2B) */
    printf("\nTekst: \"%s\"\n", polish);
    printf("strlen: %zu bajtow (nie 3 znaki!)\n", strlen(polish));  /* 5 bajtów */
    printf("Bajty: ");
    wypisz_bajty(polish);
    /* wypisze: 6F C5 82 C3 B3  (o + ł(2B) + ó(2B)) */

    /* ---- Ręczne kodowanie UTF-8 ---- */
    printf("\n=== Reczne kodowanie UTF-8 ===\n\n");

    uint8_t wynik[5];  /* bufor na zakodowany znak (max 4 bajty + zapas) */
    int n;             /* liczba zakodowanych bajtów */

    /* Kodujemy U+0041 = 'A' – 1 bajt */
    n = koduj_utf8(0x0041, wynik);
    printf("U+0041 (A): %d bajt(y): ", n);
    for (int i = 0; i < n; i++) printf("%02X ", wynik[i]);
    printf("\n");  /* wypisze: 41 */

    /* Kodujemy U+00F3 = 'ó' – 2 bajty */
    n = koduj_utf8(0x00F3, wynik);
    printf("U+00F3 (ó): %d bajt(y): ", n);
    for (int i = 0; i < n; i++) printf("%02X ", wynik[i]);
    printf("\n");  /* wypisze: C3 B3 */

    /* Kodujemy U+0142 = 'ł' – 2 bajty */
    n = koduj_utf8(0x0142, wynik);
    printf("U+0142 (ł): %d bajt(y): ", n);
    for (int i = 0; i < n; i++) printf("%02X ", wynik[i]);
    printf("\n");  /* wypisze: C5 82 */

    /* Kodujemy U+20AC = '€' – 3 bajty */
    n = koduj_utf8(0x20AC, wynik);
    printf("U+20AC (€): %d bajt(y): ", n);
    for (int i = 0; i < n; i++) printf("%02X ", wynik[i]);
    printf("\n");  /* wypisze: E2 82 AC */

    /* ---- Analiza struktury bajtu UTF-8 ---- */
    printf("\n=== Rozpoznawanie roli bajtu w UTF-8 ===\n\n");

    uint8_t bajty[] = {0x41, 0xC3, 0xB3, 0xE2, 0x82, 0xAC};
    /* A(1B)      ó(2B):bajt1  ó:bajt2  €(3B):b1   €:b2   €:b3  */

    for (size_t i = 0; i < sizeof(bajty); i++) {
        uint8_t b = bajty[i];
        printf("Bajt 0x%02X (%d): ", b, b);

        if ((b & 0x80) == 0x00) {             /* 0xxxxxxx */
            printf("1-bajtowy ASCII (U+%04X)\n", (uint32_t)b);
        } else if ((b & 0xE0) == 0xC0) {      /* 110xxxxx */
            printf("1. bajt sekwencji 2-bajtowej\n");
        } else if ((b & 0xF0) == 0xE0) {      /* 1110xxxx */
            printf("1. bajt sekwencji 3-bajtowej\n");
        } else if ((b & 0xF8) == 0xF0) {      /* 11110xxx */
            printf("1. bajt sekwencji 4-bajtowej\n");
        } else if ((b & 0xC0) == 0x80) {      /* 10xxxxxx */
            printf("bajt kontynuacji (nie-poczatkowy)\n");
        } else {
            printf("NIEPOPRAWNY bajt UTF-8\n");
        }
    }

    return 0;
}
```

## 4. UTF-16

### Teoria

**UTF-16** używa 16-bitowych jednostek kodu (ang. *code unit*). Większość znaków zajmuje jedną jednostkę (2 bajty), ale znaki spoza BMP (Plane 0, tj. U+10000 i wyżej) potrzebują dwóch jednostek – tzw. **surrogate pair** (para zastępcza).

**BMP (Basic Multilingual Plane):** znaki U+0000–U+FFFF, z wyjątkiem zakresu U+D800–U+DFFF (surrogate range). Kodowanie: bezpośrednio jako 16-bitowa wartość.

**Surrogate pairs dla U+10000–U+10FFFF:**
- Odejmujemy 0x10000 od punktu kodowego, dostajemy 20 bitów
- Górne 10 bitów: dodajemy 0xD800 → *high surrogate* (0xD800–0xDBFF)
- Dolne 10 bitów: dodajemy 0xDC00 → *low surrogate* (0xDC00–0xDFFF)

**Endianness (kolejność bajtów):**
- UTF-16 LE (Little Endian): najpierw młodszy bajt – `A` = `41 00`
- UTF-16 BE (Big Endian): najpierw starszy bajt – `A` = `00 41`
- BOM (Byte Order Mark) U+FEFF na początku pliku informuje o kolejności

**Gdzie się używa:** Windows API, Java, JavaScript (String), .NET. Windows wewnętrznie używa UTF-16 LE.

### Kod C

```c
#include <stdio.h>      /* printf */
#include <stdint.h>     /* uint16_t, uint32_t */
#include <locale.h>     /* setlocale */

/* Struktura surrogate pair */
typedef struct {
    uint16_t high;   /* wysoki surogat (0xD800-0xDBFF) */
    uint16_t low;    /* niski surogat  (0xDC00-0xDFFF) */
} SurrogatePair;

/* Kodowanie punktu kodowego U+10000-U+10FFFF na surrogate pair */
SurrogatePair koduj_surrogate(uint32_t codepoint) {
    SurrogatePair sp;
    uint32_t offset = codepoint - 0x10000;  /* odejmujemy 0x10000 */
    sp.high = (uint16_t)(0xD800 | (offset >> 10));       /* górne 10 bitów */
    sp.low  = (uint16_t)(0xDC00 | (offset & 0x3FF));     /* dolne 10 bitów */
    return sp;
}

/* Dekodowanie surrogate pair na punkt kodowy */
uint32_t dekoduj_surrogate(uint16_t high, uint16_t low) {
    /* Sprawdzamy czy to naprawdę surrogaty */
    if (high < 0xD800 || high > 0xDBFF) return 0;  /* błąd – zły high surrogate */
    if (low  < 0xDC00 || low  > 0xDFFF) return 0;  /* błąd – zły low surrogate  */

    uint32_t offset = ((uint32_t)(high - 0xD800) << 10) /* górne 10 bitów */
                    | (uint32_t)(low  - 0xDC00);         /* dolne 10 bitów */
    return offset + 0x10000;  /* dodajemy z powrotem 0x10000 */
}

/* Wypisanie 16-bitowej wartości jako bajty (LE i BE) */
void wypisz_utf16(uint16_t val) {
    uint8_t lo = (uint8_t)(val & 0xFF);        /* młodszy bajt */
    uint8_t hi = (uint8_t)((val >> 8) & 0xFF); /* starszy bajt */
    printf("LE: %02X %02X  BE: %02X %02X", lo, hi, hi, lo);
}

int main(void) {
    setlocale(LC_ALL, "pl_PL.UTF-8");

    /* ---- BMP – znaki jednojednostkowe ---- */
    printf("=== UTF-16: znaki BMP (1 jednostka = 2 bajty) ===\n\n");

    /* Tablica znaków BMP z ich kodami UTF-16 */
    struct { uint32_t cp; const char *nazwa; } bmp[] = {
        {0x0041, "A           (Latin A)"},
        {0x00F3, "ó           (ó z kreska)"},
        {0x0142, "ł           (ł z kreska)"},
        {0x03B1, "α           (alfa gr.)"},
        {0x4E2D, "中          (chiński: środek)"},
        {0x20AC, "€           (euro)"},
    };

    for (size_t i = 0; i < sizeof(bmp)/sizeof(bmp[0]); i++) {
        uint16_t utf16_val = (uint16_t)bmp[i].cp; /* bezpośrednia konwersja dla BMP */
        printf("U+%04X = %s -> UTF-16: 0x%04X  (", bmp[i].cp, bmp[i].nazwa, utf16_val);
        wypisz_utf16(utf16_val);
        printf(")\n");
    }

    /* ---- Surrogate pairs – znaki poza BMP ---- */
    printf("\n=== UTF-16: znaki poza BMP (surrogate pair = 4 bajty) ===\n\n");

    struct { uint32_t cp; const char *nazwa; } poza_bmp[] = {
        {0x1F600, "😀 (emoji uśmiech)"},
        {0x1F1F5, "🇵 (flag letter P)"},
        {0x10000, "𐀀 (Linear B)"},
        {0x1D11E, "𝄞 (G-klucz muzyczny)"},
    };

    for (size_t i = 0; i < sizeof(poza_bmp)/sizeof(poza_bmp[0]); i++) {
        SurrogatePair sp = koduj_surrogate(poza_bmp[i].cp);
        printf("U+%05X = %s\n", poza_bmp[i].cp, poza_bmp[i].nazwa);
        printf("  High surrogate: U+%04X (0x%04X)  ", sp.high, sp.high);
        wypisz_utf16(sp.high);
        printf("\n");
        printf("  Low  surrogate: U+%04X (0x%04X)  ", sp.low, sp.low);
        wypisz_utf16(sp.low);
        printf("\n");

        /* Weryfikacja – dekodujemy z powrotem */
        uint32_t decoded = dekoduj_surrogate(sp.high, sp.low);
        printf("  Weryfikacja: U+%05X %s\n\n",
               decoded,
               decoded == poza_bmp[i].cp ? "(OK)" : "(BLAD!)");
    }

    /* ---- Identyfikacja jednostki UTF-16 ---- */
    printf("=== Rozpoznawanie jednostek UTF-16 ===\n\n");

    uint16_t jednostki[] = {0x0041, 0x00F3, 0xD83D, 0xDE00, 0x4E2D};
    /*           A           ó       high surr. low surr.  中   */

    for (size_t i = 0; i < sizeof(jednostki)/sizeof(jednostki[0]); i++) {
        uint16_t u = jednostki[i];
        printf("0x%04X: ", u);

        if (u >= 0xD800 && u <= 0xDBFF) {
            printf("HIGH surrogate (pierwsza jednostka pary)\n");
        } else if (u >= 0xDC00 && u <= 0xDFFF) {
            printf("LOW surrogate (druga jednostka pary)\n");
        } else {
            printf("Normalny znak BMP (U+%04X)\n", (uint32_t)u);
        }
    }

    /* ---- BOM – Byte Order Mark ---- */
    printf("\n=== Byte Order Mark (BOM) ===\n\n");
    uint16_t bom = 0xFEFF;  /* BOM to U+FEFF */
    printf("BOM = U+FEFF = 0x%04X\n", bom);
    printf("UTF-16 LE BOM: ");
    wypisz_utf16(bom);      /* FF FE – little endian */
    printf("\n");
    /* Jeśli plik zaczyna się od FF FE -> UTF-16 LE */
    /* Jeśli plik zaczyna się od FE FF -> UTF-16 BE */

    return 0;
}
```

## 5. UTF-32

### Teoria

**UTF-32** używa stałej długości 32-bitowych jednostek. Każdy znak Unicode to dokładnie jedna jednostka (4 bajty), niezależnie od wartości punktu kodowego.

**Zalety:**
- Prosta implementacja – `s[i]` to i-ty znak, nie i-ty bajt
- Indeksowanie w czasie stałym O(1)
- Nie ma problemów z surrogates ani wielobajtowymi sekwencjami

**Wady:**
- Zużywa 4× więcej pamięci niż UTF-8 dla tekstów angielskich
- Rzadko używany w praktyce na dysku/sieci
- Kompatybilność z ASCII: tak, ale wartości 0–127 zajmują 4 bajty zamiast 1

**Zakres:** wszystkie punkty U+0000–U+10FFFF mieszczą się w 21 bitach, ale UTF-32 rezerwuje pełne 32 bity. Pozostałe 11 bitów jest ustawione na 0.

**Użycie w C:** `char32_t` z nagłówka `uchar.h`, literały z prefiksem `U"..."`.

### Kod C

```c
#include <stdio.h>      /* printf */
#include <stdint.h>     /* uint32_t */
#include <uchar.h>      /* char32_t */
#include <string.h>     /* dla pomocniczych operacji */
#include <locale.h>     /* setlocale */

/* Funkcja obliczająca długość stringa UTF-32 (liczba znaków) */
size_t strlen32(const char32_t *s) {
    size_t len = 0;             /* licznik długości */
    while (s[len] != U'\0') {   /* U'\0' = null terminator dla char32_t */
        len++;                   /* każdy element to jeden pełny znak */
    }
    return len;
}

/* Wypisanie stringa UTF-32 jako sekwencji punktów kodowych */
void wypisz_codepoints32(const char32_t *s) {
    for (size_t i = 0; s[i] != U'\0'; i++) {
        printf("U+%04X ", (uint32_t)s[i]);  /* każdy element to punkt kodowy */
    }
    printf("\n");
}

/* Wypisanie stringa UTF-32 bajt po bajcie (LE) */
void wypisz_bajty32(const char32_t *s) {
    for (size_t i = 0; s[i] != U'\0'; i++) {
        uint32_t cp = (uint32_t)s[i];
        /* UTF-32 LE: 4 bajty w kolejności od najmłodszego */
        printf("[%02X %02X %02X %02X] ",
               (unsigned)(cp & 0xFF),          /* bajt 0 (najmłodszy) */
               (unsigned)((cp >> 8) & 0xFF),   /* bajt 1 */
               (unsigned)((cp >> 16) & 0xFF),  /* bajt 2 */
               (unsigned)((cp >> 24) & 0xFF)); /* bajt 3 (najstarszy) */
    }
    printf("\n");
}

int main(void) {
    setlocale(LC_ALL, "pl_PL.UTF-8");

    /* ---- Tworzenie stringów UTF-32 ---- */
    printf("=== UTF-32: stałe 4 bajty na znak ===\n\n");

    /* Literał UTF-32 – prefiks U"..." */
    const char32_t *s1 = U"Ala";         /* 3 znaki × 4 bajty + 4 bajty null */
    const char32_t *s2 = U"ołó";         /* 3 znaki × 4 bajty + 4 bajty null */

    /* Rozmiar w bajtach = (liczba_znaków + 1) × 4 */
    printf("String \"%s\":\n", "Ala");
    printf("  Liczba znakow: %zu\n", strlen32(s1));
    printf("  Rozmiar w pamieci: %zu bajtow\n", (strlen32(s1) + 1) * sizeof(char32_t));
    /* wypisze: 3 znaki, 16 bajtów (3 znaki × 4B + null × 4B) */
    printf("  Punkty kodowe: ");
    wypisz_codepoints32(s1);
    printf("  Bajty (UTF-32 LE): ");
    wypisz_bajty32(s1);

    printf("\nString \"%s\":\n", "ołó");
    printf("  Liczba znakow: %zu\n", strlen32(s2));
    printf("  Rozmiar w pamieci: %zu bajtow\n", (strlen32(s2) + 1) * sizeof(char32_t));
    /* wypisze: 3 znaki, 16 bajtów – UTF-32 zawsze 4 bajty/znak! */
    printf("  Punkty kodowe: ");
    wypisz_codepoints32(s2);
    printf("  Bajty (UTF-32 LE): ");
    wypisz_bajty32(s2);

    /* ---- Porównanie rozmiarów UTF-8 vs UTF-32 ---- */
    printf("\n=== Porownanie rozmiarow UTF-8 vs UTF-32 ===\n\n");

    /* Ten sam tekst w różnych kodowaniach */
    const char     *utf8  = "Hello";   /* UTF-8:  5 × 1B = 5B */
    const char32_t *utf32 = U"Hello";  /* UTF-32: 5 × 4B = 20B */
    printf("\"Hello\" w UTF-8:  %zu bajtow\n", strlen(utf8));         /* 5 */
    printf("\"Hello\" w UTF-32: %zu bajtow\n", strlen32(utf32) * 4);  /* 20 */

    const char     *utf8_pl  = "Zażółć";   /* UTF-8: więcej bajtów dla polskich liter */
    const char32_t *utf32_pl = U"Zażółć";  /* UTF-32: zawsze 4B/znak */
    printf("\n\"Zażółć\" w UTF-8:  %zu bajtow\n", strlen(utf8_pl));
    /* Z=1, a=1, ż=2, ó=2, ł=2, ć=2 → razem 10 bajtów */
    printf("\"Zażółć\" w UTF-32: %zu bajtow\n", strlen32(utf32_pl) * 4);
    /* 6 znaków × 4 bajty = 24 bajty */

    /* ---- Bezpośredni dostęp do i-tego znaku ---- */
    printf("\n=== Bezposredni dostep do i-tego znaku ===\n\n");

    const char32_t *tekst = U"Ala ma kota";
    printf("Tekst: \"Ala ma kota\"\n");
    printf("Liczba znakow: %zu\n", strlen32(tekst));  /* 11 */

    /* W UTF-32 indeksowanie jest proste: tekst[i] = i-ty znak */
    printf("Znak [0]: U+%04X\n", (uint32_t)tekst[0]);  /* U+0041 = A */
    printf("Znak [4]: U+%04X\n", (uint32_t)tekst[4]);  /* U+006D = m */
    printf("Znak [7]: U+%04X\n", (uint32_t)tekst[7]);  /* U+006B = k */
    /* To NIE działa tak prosto w UTF-8 czy UTF-16! */

    /* ---- sizeof a długość stringa ---- */
    printf("\n=== sizeof vs strlen32 ===\n\n");

    char32_t lokalny[] = U"ABC";  /* tablica na stosie */
    printf("sizeof(lokalny):      %zu bajtow\n", sizeof(lokalny));
    /* sizeof zwraca całkowity rozmiar tablicy: 4 znaki × 4B = 16B (łącznie z null) */
    printf("strlen32(lokalny):    %zu znakow\n", strlen32(lokalny));
    /* strlen32 zwraca liczbę znaków bez null: 3 */
    printf("Rozmiar bez terminatora: %zu bajtow\n", strlen32(lokalny) * sizeof(char32_t));
    /* 3 × 4 = 12 */

    return 0;
}
```

## 6. Typy znakowe w C

### Teoria

C oferuje kilka typów do przechowywania znaków i łańcuchów:

**`char`** – podstawowy typ znakowy, zwykle 8-bitowy. Może być `signed char` lub `unsigned char` (zależy od platformy). Domyślne kodowanie zależy od ustawień systemu (ASCII, lokalne 8-bitowe kodowanie). Starczy do kodowania ASCII i podstawowych kodowań europejskich w 1 bajcie.

**`char8_t`** (C23) – gwarantuje 8 bitów bez znaku (`unsigned char`). Przeznaczony dla UTF-8. Literały: `u8"tekst"`.

**`char16_t`** (C11, z `uchar.h`) – co najmniej 16 bitów. Przeznaczony dla UTF-16. Literały: `u"tekst"`.

**`char32_t`** (C11, z `uchar.h`) – co najmniej 32 bity. Przeznaczony dla UTF-32. Literały: `U"tekst"`.

**`wchar_t`** – „szeroki znak". Zależy od platformy: 16 bitów na Windows, 32 bity na Linux/macOS. Literały: `L"tekst"`. Wprowadzony w C95. **Nie jest powiązany z konkretnym kodowaniem** – to platforma decyduje.

**Kluczowa różnica `sizeof` vs `strlen`:**
- `sizeof(tablica)` – rozmiar całej tablicy w bajtach (łącznie z terminatorem `'\0'`)
- `strlen(tablica)` – liczba znaków do terminatora (bez `'\0'`)

### Kod C

```c
#include <stdio.h>      /* printf */
#include <string.h>     /* strlen */
#include <uchar.h>      /* char8_t, char16_t, char32_t – od C11 */
#include <wchar.h>      /* wchar_t, wcslen */
#include <locale.h>     /* setlocale */

/* Funkcja wypisująca bajty dowolnego obszaru pamięci */
void wypisz_bajty_mem(const void *ptr, size_t n) {
    const unsigned char *p = (const unsigned char *)ptr;  /* rzutowanie na bajty */
    for (size_t i = 0; i < n; i++) {
        printf("%02X ", p[i]);  /* każdy bajt jako hex */
    }
    printf("\n");
}

int main(void) {
    setlocale(LC_ALL, "pl_PL.UTF-8");  /* locale UTF-8 */

    /* ---- sizeof dla typów znakowych ---- */
    printf("=== Rozmiary typow znakowych ===\n\n");

    printf("sizeof(char)     = %zu bajt\n",  sizeof(char));
    printf("sizeof(char8_t)  = %zu bajt\n",  sizeof(char8_t));
    printf("sizeof(char16_t) = %zu bajty\n", sizeof(char16_t));
    printf("sizeof(char32_t) = %zu bajty\n", sizeof(char32_t));
    printf("sizeof(wchar_t)  = %zu bajty (16 na Win, 32 na Linux)\n", sizeof(wchar_t));

    /* ---- Deklaracje łańcuchów ---- */
    printf("\n=== Deklaracje lancuchow znakowych ===\n\n");

    /* char[] – klasyczny sposób, literał bez prefiksu */
    char s_char[] = "Wielka niedzwiedzica";  /* array of char, null-terminated */
    printf("char[]:   \"%s\"\n", s_char);
    printf("  sizeof:  %zu bajtow (z '\\0')\n", sizeof(s_char));  /* 21 */
    printf("  strlen:  %zu znakow (bez '\\0')\n", strlen(s_char)); /* 20 */

    /* char[] – inicjalizacja znakami osobno */
    char ktos[] = {'M', 'i', 'k', 'o', 'l', 'a', 'j', '\0'};  /* ręczne null */
    printf("\nchar ktos[]: \"%s\"\n", ktos);
    printf("  sizeof:  %zu bajtow\n", sizeof(ktos));  /* 8 (7 znaków + '\0') */
    printf("  strlen:  %zu znakow\n", strlen(ktos));  /* 7 */
    /* sizeof != strlen – to jest ważne! */
    printf("  sizeof(%zu) != strlen(%zu)\n", sizeof(ktos), strlen(ktos));

    /* char[] – deklaracja bufora (niezainicjalizowanego) */
    char imie[20];   /* bufor na 19 znaków + '\0' */
    printf("\nchar imie[20]: bufor na max %zu znakow\n", sizeof(imie) - 1);

    /* char8_t – UTF-8 (C23) */
    /* Uwaga: char8_t nie jest bezpośrednio obsługiwany przez printf %s */
    /* Rzutujemy na const char* dla printf                                */
    char8_t s_u8[] = u8"Tekst UTF-8: ąęó";   /* prefiks u8, C23 */
    printf("\nchar8_t (u8\"...\"):\n");
    printf("  Bajty: ");
    wypisz_bajty_mem(s_u8, sizeof(s_u8) - 1);  /* bez null terminatora */
    printf("  sizeof:   %zu bajtow\n", sizeof(s_u8));
    /* sizeof > strlen dla polskich liter bo każda polska litera = 2 bajty */

    /* char16_t – UTF-16 */
    char16_t s_u16[] = u"Ala ó";   /* prefiks u, UTF-16 */
    printf("\nchar16_t (u\"...\"):\n");
    printf("  sizeof:   %zu bajtow\n", sizeof(s_u16));  /* (5+1) * 2 = 12 */
    printf("  Jednostki kodu: %zu\n", sizeof(s_u16)/sizeof(char16_t) - 1);
    printf("  Bajty: ");
    wypisz_bajty_mem(s_u16, sizeof(s_u16) - sizeof(char16_t));

    /* char32_t – UTF-32 */
    char32_t s_u32[] = U"Ala ó";   /* prefiks U, UTF-32 */
    printf("\nchar32_t (U\"...\"):\n");
    printf("  sizeof:   %zu bajtow\n", sizeof(s_u32));  /* (5+1) * 4 = 24 */
    printf("  Znaki:    %zu\n", sizeof(s_u32)/sizeof(char32_t) - 1);
    printf("  Bajty: ");
    wypisz_bajty_mem(s_u32, sizeof(s_u32) - sizeof(char32_t));

    /* wchar_t – szerokie znaki */
    wchar_t s_wchar[] = L"Ala ó";  /* prefiks L, wchar_t */
    printf("\nwchar_t (L\"...\"):\n");
    printf("  sizeof:   %zu bajtow\n", sizeof(s_wchar));  /* (5+1) * sizeof(wchar_t) */
    printf("  wcslen:   %zu znakow\n", wcslen(s_wchar));   /* 5 – wcslen dla wchar_t */
    printf("  Bajty: ");
    wypisz_bajty_mem(s_wchar, wcslen(s_wchar) * sizeof(wchar_t));

    /* ---- Wskaźniki vs tablice ---- */
    printf("\n=== Wskaznik vs tablica ===\n\n");

    char tablica[] = "Hello";      /* tablica znaków – modyfikowalna */
    char *wskaznik = "Hello";      /* wskaźnik do literału – NIE modyfikowalny! */

    printf("tablica: sizeof=%zu, strlen=%zu\n",
           sizeof(tablica), strlen(tablica));  /* sizeof=6 (z '\0'), strlen=5 */
    printf("wskaznik: sizeof=%zu (rozmiar wskaznika!), strlen=%zu\n",
           sizeof(wskaznik), strlen(wskaznik));  /* sizeof=8 (64bit) lub 4 (32bit), strlen=5 */
    /* UWAGA: sizeof(wskaznik) to rozmiar wskaźnika, NIE stringa! */

    /* ---- Modyfikowalność ---- */
    tablica[0] = 'h';  /* OK – tablica jest modyfikowalna */
    printf("Tablica po modyfikacji: \"%s\"\n", tablica);

    /* wskaznik[0] = 'h';  // BŁĄD WYKONANIA – literał jest read-only! */
    printf("Wskaznik: modyfikacja literału = undefined behavior!\n");

    return 0;
}
```

## 7. Operacje arytmetyczne na znakach

### Teoria

Znaki w C to po prostu małe liczby całkowite. Można na nich wykonywać operacje arytmetyczne. To szczególnie użyteczne z ASCII, gdzie litery i cyfry tworzą ciągłe zakresy.

**Kluczowe właściwości ASCII:**
- `'A'` = 65, `'B'` = 66, ..., `'Z'` = 90 (różnica między literami: 1)
- `'a'` = 97, `'b'` = 98, ..., `'z'` = 122 (różnica między literami: 1)
- `'a'` - `'A'` = 32 (różnica między małą a wielką literą: zawsze 32)
- `'0'` = 48, `'1'` = 49, ..., `'9'` = 57 (różnica między cyframi: 1)

**Zastosowania:**
- `znakx - 'a'` → numer litery w alfabecie (a=0, b=1, ..., z=25)
- `znaky - '0'` → wartość cyfry jako liczba (np. `'7' - '0'` = 7)
- `c | 0x20` → konwersja na małą literę (szybsza niż `tolower`)
- `c & ~0x20` → konwersja na wielką literę (szybsza niż `toupper`)

### Kod C

```c
#include <stdio.h>   /* printf */
#include <ctype.h>   /* toupper, tolower – dla porównania */

int main(void) {
    /* ---- Konwersja znaku na numer w alfabecie ---- */
    printf("=== Numer litery w alfabecie ===\n\n");

    char litera;
    /* Iterujemy po literach i obliczamy ich pozycję */
    for (litera = 'a'; litera <= 'z'; litera++) {
        int pozycja = litera - 'a';  /* a→0, b→1, ..., z→25 */
        printf("'%c' - 'a' = %d\n", litera, pozycja);
    }

    /* ---- Konwersja cyfry-znaku na wartość liczbową ---- */
    printf("\n=== Cyfra-znak na liczbe ===\n\n");

    for (char cyfra = '0'; cyfra <= '9'; cyfra++) {
        int wartosc = cyfra - '0';  /* '0'→0, '1'→1, ..., '9'→9 */
        printf("'%c' - '0' = %d\n", cyfra, wartosc);
    }

    /* ---- Konwersja wartości na cyfrę-znak ---- */
    printf("\n=== Liczba na cyfre-znak ===\n\n");
    for (int v = 0; v <= 9; v++) {
        char c = (char)(v + '0');  /* 0→'0', 1→'1', ..., 9→'9' */
        printf("%d + '0' = '%c'\n", v, c);
    }

    /* ---- Konwersja wielka/mała litera ---- */
    printf("\n=== Konwersja wielkosci liter (arytmetycznie) ===\n\n");

    for (char c = 'A'; c <= 'Z'; c++) {
        char mala = (char)(c + 32);   /* wielka → mała: dodajemy 32 */
        char wielka_z_malej = (char)(c + 32 - 32);  /* z powrotem */
        printf("'%c' + 32 = '%c'\n", c, mala);
    }

    /* Alternatywnie: operacje bitowe */
    printf("\n=== Konwersja bitowa (szybsza) ===\n\n");
    char t1 = 'A';
    char t2 = 'z';
    printf("'A' | 0x20 = '%c' (na mała)\n", (char)(t1 | 0x20));   /* A → a */
    printf("'z' & ~0x20 = '%c' (na wielką)\n", (char)(t2 & ~0x20)); /* z → Z */
    /* Uwaga: to działa TYLKO dla liter ASCII a-z / A-Z! */

    /* ---- Kodowanie szesnastkowe ---- */
    printf("\n=== Konwersja cyfra hex na wartosc ===\n\n");
    const char *hex_cyfry = "0123456789ABCDEF";
    for (int i = 0; hex_cyfry[i] != '\0'; i++) {
        char c = hex_cyfry[i];
        int val;
        if (c >= '0' && c <= '9') {
            val = c - '0';           /* cyfra: 0–9 */
        } else if (c >= 'A' && c <= 'F') {
            val = c - 'A' + 10;      /* litera: A=10, B=11, ..., F=15 */
        } else {
            val = -1;  /* nie powinno się zdarzyć */
        }
        printf("'%c' = %d (hex)\n", c, val);
    }

    /* ---- Przykład praktyczny: konwersja stringa na liczbę ---- */
    printf("\n=== Reczna konwersja stringa na int ===\n\n");

    const char *napis = "12345";
    int liczba = 0;  /* wynik */
    for (int i = 0; napis[i] != '\0'; i++) {
        if (napis[i] >= '0' && napis[i] <= '9') {          /* sprawdzamy czy cyfra */
            liczba = liczba * 10 + (napis[i] - '0');        /* przesuwamy i dodajemy */
        }
    }
    printf("String \"%s\" jako int: %d\n", napis, liczba);  /* wypisze: 12345 */

    /* ---- ROT13 – przykład szyfrowania ---- */
    printf("\n=== ROT13 (obrot o 13 pozycji) ===\n\n");

    const char *tekst = "Hello World";
    printf("Oryginal: \"%s\"\nROT13:    \"", tekst);
    for (int i = 0; tekst[i] != '\0'; i++) {
        char c = tekst[i];
        if (c >= 'a' && c <= 'z') {
            /* Mała litera: obracamy w zakresie a-z */
            c = (char)('a' + (c - 'a' + 13) % 26);
        } else if (c >= 'A' && c <= 'Z') {
            /* Wielka litera: obracamy w zakresie A-Z */
            c = (char)('A' + (c - 'A' + 13) % 26);
        }
        printf("%c", c);  /* spacje i inne znaki bez zmian */
    }
    printf("\"\n");  /* wypisze: "Uryyb Jbeyq" */

    return 0;
}
```

## 8. Biblioteka standardowa

### Teoria

C dostarcza kilka nagłówków do pracy z tekstem:

**`<string.h>`** – funkcje dla `char`: kopiowanie, konkatenacja, porównanie, wyszukiwanie, tokenizacja. **Tylko dla `char`**, nie dla `wchar_t`.

**`<ctype.h>`** – klasyfikacja i konwersja znaków `char`: `isalpha`, `isdigit`, `toupper` itp. **Tylko dla `char`**.

**`<uchar.h>`** (C11) – konwersja między `char`, `char16_t`, `char32_t`. Przydatne gdy trzeba konwertować między kodowaniami.

**`<wchar.h>`** – wersje funkcji z `<string.h>` dla `wchar_t`: `wcslen`, `wcscpy`, `wcscat` itp. Wybrane operacje na UTF (na ile `wchar_t` = Unicode, zależy od platformy).

**`<wctype.h>`** – odpowiednik `<ctype.h>` dla `wchar_t`: `iswalpha`, `towupper` itp.

**Zewnętrzne biblioteki Unicode:** Standardowa biblioteka C ma ograniczone wsparcie dla pełnego Unicode. Do poważnej pracy z Unicode używa się bibliotek jak ICU, utf8proc, libunistring.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strlen, strcpy, strcat, strcmp, strchr, strstr */
#include <ctype.h>    /* isalpha, isdigit, toupper, tolower */
#include <wchar.h>    /* wchar_t, wcslen, wcscpy */
#include <wctype.h>   /* iswalpha, towupper */
#include <locale.h>   /* setlocale */

int main(void) {
    setlocale(LC_ALL, "pl_PL.UTF-8");

    /* ---- <string.h> – podstawowe użycie ---- */
    printf("=== <string.h> ===\n\n");

    char s1[100] = "Hello";       /* bufor z miejscem */
    char s2[] = " World";         /* string do dołączenia */

    printf("strlen(\"%s\"): %zu\n", s1, strlen(s1));  /* 5 */
    strcat(s1, s2);                /* konkatenacja: s1 = "Hello World" */
    printf("Po strcat: \"%s\"\n", s1);

    /* ---- <ctype.h> – klasyfikacja znaków ---- */
    printf("\n=== <ctype.h> ===\n\n");

    char test_chars[] = "aB3! \t\n";   /* różne typy znaków */
    for (int i = 0; test_chars[i] != '\0'; i++) {
        char c = test_chars[i];
        printf("'%c' (kod %3d): ", c >= 32 ? c : '.', (unsigned char)c);
        /* Używamy (unsigned char) bo isalpha itp. wymaga unsigned char lub EOF */
        if (isalpha((unsigned char)c))  printf("litera ");
        if (isdigit((unsigned char)c))  printf("cyfra ");
        if (isupper((unsigned char)c))  printf("wielka ");
        if (islower((unsigned char)c))  printf("mala ");
        if (isspace((unsigned char)c))  printf("biale ");
        if (ispunct((unsigned char)c))  printf("interpunkcja ");
        printf("\n");
    }

    /* ---- <wchar.h> – szerokie znaki ---- */
    printf("\n=== <wchar.h> ===\n\n");

    wchar_t ws1[100] = L"Cześć";   /* wchar_t string z polskimi literami */
    printf("wcslen(L\"Cześć\"): %zu\n", wcslen(ws1));  /* 5 – poprawna liczba znaków */
    /* Dla wchar_t printf używa %ls */
    printf("String: %ls\n", ws1);

    /* ---- <wctype.h> – klasyfikacja szerokich znaków ---- */
    printf("\n=== <wctype.h> ===\n\n");

    wchar_t wtest[] = L"Ala ąęó 123";
    for (int i = 0; wtest[i] != L'\0'; i++) {
        wchar_t wc = wtest[i];
        printf("U+%04X ('%lc'): ", (unsigned)wc, wc);
        if (iswalpha(wc))  printf("litera ");   /* rozpoznaje polskie litery! */
        if (iswdigit(wc))  printf("cyfra ");
        if (iswupper(wc))  printf("wielka ");
        if (iswlower(wc))  printf("mala ");
        if (iswspace(wc))  printf("biale ");
        printf("\n");
    }

    /* Porównanie ctype vs wctype dla polskiej litery */
    printf("\n=== ctype vs wctype dla 'ą' ===\n\n");
    /* W UTF-8 'ą' to 2 bajty: 0xC4, 0x85 */
    /* isalpha((unsigned char)'ą') – widzi tylko jeden bajt! */
    /* iswalpha(L'ą') – widzi pełny wchar_t, działa poprawnie */
    printf("isalpha  dla bajtu 0xC4 (1. bajt ą w UTF-8): %d\n",
           isalpha(0xC4));   /* może dać nieoczekiwany wynik */
    printf("iswalpha dla L'ą' (wchar_t): %d\n",
           iswalpha(L'\u0105'));  /* 1 (prawda) – poprawnie */

    return 0;
}
```

## 9. `<string.h>` – kopiowanie

### Teoria

Funkcje kopiujące z `<string.h>` przenoszą zawartość jednego łańcucha do drugiego:

**`strcpy(dst, src)`** – kopiuje `src` do `dst` włącznie z `'\0'`. **Niebezpieczna** – nie sprawdza rozmiaru `dst`. Jeśli `src` jest dłuższe niż `dst`, następuje przepełnienie bufora.

**`strncpy(dst, src, n)`** – kopiuje maksymalnie `n` znaków. Jeśli `src` krótsze, dopełnia `dst` zerami do `n`. **Uwaga:** jeśli `src` ma ≥ `n` znaków, `dst` nie będzie miał `'\0'` na końcu!

**`strcpy_s(dst, dstmax, src)`** (Annex K) – bezpieczna wersja, sprawdza rozmiar. Zwraca `errno_t`.

**`strncpy_s(dst, dstmax, src, n)`** – bezpieczna wersja `strncpy`.

**`strdup(src)`** – alokuje nowy bufor na stercie (`malloc`), kopiuje tam `src`, zwraca wskaźnik. **Trzeba zwolnić przez `free()`!**

**`strndup(src, n)`** – jak `strdup`, ale kopiuje co najwyżej `n` znaków.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strcpy, strncpy, strdup, strndup */
#include <stdlib.h>   /* free – do zwalniania pamięci po strdup */

int main(void) {
    /* ---- strcpy ---- */
    printf("=== strcpy ===\n\n");

    char src[] = "Hello World";   /* źródło */
    char dst[100];                 /* docelowy bufor – duży, bezpieczny */

    char *ret = strcpy(dst, src);  /* kopiuje src do dst, zwraca dst */
    printf("src:     \"%s\"\n", src);
    printf("dst:     \"%s\"\n", dst);
    printf("ret==dst: %s\n", ret == dst ? "tak (ten sam wskaznik)" : "nie");
    /* ret wskazuje na dst */

    /* Demonstracja: strcpy kopiuje też '\0' */
    char maly[6];
    strcpy(maly, "Hello");   /* "Hello" = 5 znaków + '\0' = 6 bajtów, pasuje */
    printf("\nmaly: \"%s\" (strlen=%zu, sizeof=%zu)\n",
           maly, strlen(maly), sizeof(maly));

    /* ---- strncpy ---- */
    printf("\n=== strncpy ===\n\n");

    char bufor[10];

    /* Przypadek 1: src krótsze niż n – dopełnia zerami */
    strncpy(bufor, "Hi", 10);  /* kopiuje H, i, '\0', a potem 7 bajtów '\0' */
    printf("strncpy(bufor, \"Hi\", 10): \"%s\"\n", bufor);
    printf("Bajty bufora: ");
    for (int i = 0; i < 10; i++) printf("%02X ", (unsigned char)bufor[i]);
    printf("\n");
    /* wypisze: 48 69 00 00 00 00 00 00 00 00 */

    /* Przypadek 2: src dłuższe niż n – BRAK null terminatora! */
    char niebezpieczny[5];
    strncpy(niebezpieczny, "Hello World", 5);  /* kopiuje tylko "Hello", bez '\0'! */
    niebezpieczny[4] = '\0';  /* MUSIMY ręcznie dodać '\0'! */
    printf("strncpy(niebezpieczny, \"Hello World\", 5): \"%s\"\n", niebezpieczny);
    /* bez ręcznego '\0' – undefined behavior! */

    /* ---- strdup ---- */
    printf("\n=== strdup ===\n\n");

    const char *oryginal = "Programowanie w C";
    char *kopia = strdup(oryginal);  /* alokuje pamięć i kopiuje */

    if (kopia == NULL) {             /* zawsze sprawdzaj NULL! */
        printf("Blad: brak pamieci!\n");
        return 1;
    }

    printf("oryginal: \"%s\" (adres: %p)\n", oryginal, (void*)oryginal);
    printf("kopia:    \"%s\" (adres: %p)\n", kopia,    (void*)kopia);
    printf("Inne adresy: %s\n", oryginal != kopia ? "tak (oddzielna pamiec)" : "nie");

    /* Modyfikacja kopii nie wpływa na oryginał */
    kopia[0] = 'p';  /* zmiana pierwszej litery kopii */
    printf("\nPo modyfikacji kopii:\n");
    printf("oryginal: \"%s\"\n", oryginal);  /* bez zmian */
    printf("kopia:    \"%s\"\n", kopia);     /* zmieniona */

    free(kopia);   /* OBOWIĄZKOWE – zwalniamy pamięć po strdup */
    kopia = NULL;  /* dobra praktyka: ustawiamy NULL po free */

    /* ---- strndup ---- */
    printf("\n=== strndup ===\n\n");

    const char *dlugi = "Abecadło z pieca spadło";
    char *fragment = strndup(dlugi, 8);  /* kopiuje pierwsze 8 znaków */

    if (fragment == NULL) {
        printf("Blad: brak pamieci!\n");
        return 1;
    }

    printf("oryginal: \"%s\"\n", dlugi);
    printf("fragment (8 znakow): \"%s\"\n", fragment);  /* "Abecadło" (ale: ł = 2 bajty!) */
    /* Uwaga: dla UTF-8 strndup kopiuje n BAJTÓW, nie n ZNAKÓW */

    free(fragment);   /* też trzeba zwolnić! */
    fragment = NULL;

    /* ---- Różnica między tablicą a wskaźnikiem przy kopiowaniu ---- */
    printf("\n=== Przypisanie vs strcpy ===\n\n");

    char a[20] = "Ala";
    char b[20];

    /* b = a;  // ŹLE – nie można przypisać tablic w C! Błąd kompilacji */
    strcpy(b, a);  /* DOBRZE – kopiujemy zawartość */
    printf("a: \"%s\", b: \"%s\"\n", a, b);

    /* Wskaźniki: przypisanie kopiuje adres, nie zawartość */
    char *pa = a;
    char *pb = pa;   /* pb wskazuje na te same dane co pa! */
    pa[0] = 'X';    /* modyfikacja przez pa */
    printf("po pa[0]='X': pa=\"%s\", pb=\"%s\"\n", pa, pb);
    /* obydwa wypisują "Xla" – to ten sam string! */

    return 0;
}
```

## 10. `<string.h>` – konkatenacja

### Teoria

**`strcat(dst, src)`** – dołącza `src` na koniec `dst`. `dst` musi mieć wystarczająco dużo miejsca. Szuka `'\0'` w `dst`, a potem kopiuje tam `src`. **Niebezpieczna** – bez sprawdzania rozmiaru.

**`strncat(dst, src, n)`** – dołącza co najwyżej `n` znaków z `src`. Zawsze dodaje `'\0'`. Bezpieczniejsza niż `strcat`.

**`strcat_s(dst, dstmax, src)`** (Annex K) – bezpieczna wersja sprawdzająca rozmiar.

**`strncat_s(dst, dstmax, src, n)`** – bezpieczna wersja `strncat`.

**Pułapka wydajności:** wielokrotny `strcat` na tym samym buforze ma złożoność O(n²) – za każdym razem szuka końca stringa od początku. Dla wielu konkatenacji lepiej użyć wskaźnika lub `snprintf`.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strcat, strncat, strlen */

int main(void) {
    /* ---- strcat ---- */
    printf("=== strcat ===\n\n");

    char zdanie[100] = "Ala ";     /* bufor z miejscem, zaczynamy od "Ala " */

    strcat(zdanie, "ma ");          /* dołącz "ma " – szuka '\0' w zdanie, kopiuje tam */
    printf("Po 1. strcat: \"%s\"\n", zdanie);  /* "Ala ma " */

    strcat(zdanie, "kota");         /* dołącz "kota" */
    printf("Po 2. strcat: \"%s\"\n", zdanie);  /* "Ala ma kota" */

    strcat(zdanie, " i psa.");       /* dołącz " i psa." */
    printf("Po 3. strcat: \"%s\"\n", zdanie);  /* "Ala ma kota i psa." */

    printf("Dlugosc: %zu\n", strlen(zdanie));  /* 19 */

    /* Łańcuchowe strcat – zwraca wskaźnik do dst */
    char wynik[100] = "";  /* pusty string */
    /* Można łączyć wywołania: strcat(strcat(strcat(wynik, "a"), "b"), "c") */
    strcat(strcat(strcat(wynik, "a"), "b"), "c");
    printf("\nLancuchowy strcat: \"%s\"\n", wynik);  /* "abc" */

    /* ---- strncat ---- */
    printf("\n=== strncat ===\n\n");

    char buf[20] = "Start: ";   /* bufor z tekstem startowym */
    const char *dodatek = "długi tekst do dodania";

    /* Ile miejsca zostało w buforze? */
    size_t pozostalo = sizeof(buf) - strlen(buf) - 1;  /* -1 na '\0' */
    printf("Bufor: \"%s\" (strlen=%zu, pozostalo=%zu)\n",
           buf, strlen(buf), pozostalo);

    strncat(buf, dodatek, pozostalo);  /* dodajemy max tyle ile zostało */
    printf("Po strncat: \"%s\"\n", buf);
    /* strncat zawsze dodaje '\0', nawet jeśli obcina src */

    /* Przypadek gdzie strncat jest bezpieczniejszy od strcat */
    char bezpieczny[15] = "ABC";
    strncat(bezpieczny, "DEFGHIJKLMNOP", sizeof(bezpieczny) - strlen(bezpieczny) - 1);
    printf("Bezpieczny bufor[15]: \"%s\"\n", bezpieczny);
    /* doda tyle ile wejdzie, bez przepełnienia */

    /* ---- Demonstracja problemu z wydajnością O(n²) ---- */
    printf("\n=== Problem wydajnosci: O(n^2) ===\n\n");

    char duzy[1000] = "";    /* pusty bufor */
    const char *slowa[] = {"Ala", " ma", " kota", " i", " psa", NULL};

    /* Każdy strcat musi przejść cały string żeby znaleźć koniec */
    for (int i = 0; slowa[i] != NULL; i++) {
        strcat(duzy, slowa[i]);   /* O(n) każde wywołanie → O(n²) łącznie */
    }
    printf("O(n^2): \"%s\"\n", duzy);

    /* Lepsze podejście: użyj wskaźnika na koniec */
    char lepszy[1000] = "";
    char *koniec = lepszy;   /* wskaźnik na aktualne miejsce zapisu */

    for (int i = 0; slowa[i] != NULL; i++) {
        /* strcpy na wskaźnik – nie szuka końca, bo wiemy gdzie jest */
        size_t dlugosc = strlen(slowa[i]);
        memcpy(koniec, slowa[i], dlugosc);  /* kopiuj bez '\0' */
        koniec += dlugosc;                   /* przesuń wskaźnik */
    }
    *koniec = '\0';  /* dodaj '\0' na końcu */
    printf("O(n):   \"%s\"\n", lepszy);

    /* Jeszcze lepsze – snprintf: */
    char najlepszy[1000];
    snprintf(najlepszy, sizeof(najlepszy), "%s%s%s%s%s",
             "Ala", " ma", " kota", " i", " psa");
    printf("snprintf: \"%s\"\n", najlepszy);

    return 0;
}
```

```
Dlugosc: 18
```

## 11. `<string.h>` – porównywanie

### Teoria

**`strcmp(s1, s2)`** – porównuje leksykograficznie (jak w słowniku, według kodów ASCII):
- Zwraca 0 gdy `s1 == s2`
- Zwraca wartość < 0 gdy `s1` jest przed `s2` w porządku ASCII
- Zwraca wartość > 0 gdy `s1` jest za `s2`
- **Dokładna wartość (poza znakiem) nie jest określona standardem!** Tylko znak (ujemny/zero/dodatni) jest gwarantowany.

**`strncmp(s1, s2, n)`** – porównuje co najwyżej `n` znaków. Przydatne do porównywania prefiksów.

**`strcoll(s1, s2)`** – jak `strcmp`, ale uwzględnia ustawienia lokalne (`locale`). Np. po ustawieniu polskiego locale `ą` będzie przed `b`. Ważne dla sortowania tekstów w językach nieangielskich.

**Ważna pułapka:** nigdy nie używaj `==` do porównywania stringów! `s1 == s2` porównuje adresy, nie zawartość.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strcmp, strncmp, strcoll */
#include <locale.h>   /* setlocale */

/* Pomocnicza funkcja wypisująca wynik porównania czytelnie */
void pokaz_strcmp(const char *s1, const char *s2) {
    int wynik = strcmp(s1, s2);   /* porównujemy */
    printf("strcmp(\"%s\", \"%s\") = %d --> s1 %s s2\n",
           s1, s2, wynik,
           wynik < 0 ? "<" : (wynik > 0 ? ">" : "=="));
}

int main(void) {
    /* ---- strcmp ---- */
    printf("=== strcmp ===\n\n");

    pokaz_strcmp("apple", "apple");    /* 0: identyczne */
    pokaz_strcmp("apple", "banana");   /* < 0: a < b */
    pokaz_strcmp("banana", "apple");   /* > 0: b > a */
    pokaz_strcmp("abc", "abcd");       /* < 0: krótszy jest "mniejszy" */
    pokaz_strcmp("abcd", "abc");       /* > 0 */
    pokaz_strcmp("ABC", "abc");        /* < 0: wielkie litery mają niższe kody */
    pokaz_strcmp("", "");              /* 0: dwa puste stringi */
    pokaz_strcmp("", "a");             /* < 0: pusty jest "mniejszy" */

    /* Przykład porządku ASCII: wielkie litery PRZED małymi */
    printf("\nPorzadek ASCII (wielkie przed malymi):\n");
    printf("'A'=%d < 'a'=%d\n", 'A', 'a');  /* 65 < 97 */

    /* ---- Pułapka: nie używaj == do porównywania stringów! ---- */
    printf("\n=== PULAPKA: operator == ===\n\n");

    char s1[] = "Hello";
    char s2[] = "Hello";
    char *p1 = s1;
    char *p2 = s2;

    printf("s1 == s2 (porownanie adresow): %s\n",
           s1 == s2 ? "ROWNE" : "ROZNE");     /* RÓŻNE – inne adresy! */
    printf("p1 == p1 (ten sam adres): %s\n",
           p1 == p1 ? "ROWNE" : "ROZNE");     /* RÓWNE – ten sam adres! */
    printf("strcmp(s1, s2) (porownanie zawartosci): %d\n",
           strcmp(s1, s2));                    /* 0 – te same zawartości */

    /* ---- strncmp – porównanie prefiksów ---- */
    printf("\n=== strncmp ===\n\n");

    const char *url1 = "https://example.com";
    const char *url2 = "https://other.com";
    const char *url3 = "http://example.com";

    /* Sprawdzamy czy URL zaczyna się od "https://" */
    int wynik1 = strncmp(url1, "https://", 8);  /* porównujemy 8 pierwszych znaków */
    int wynik2 = strncmp(url3, "https://", 8);

    printf("URL1: \"%s\" - %s HTTPS\n", url1, wynik1 == 0 ? "jest" : "nie jest");
    printf("URL3: \"%s\" - %s HTTPS\n", url3, wynik2 == 0 ? "jest" : "nie jest");

    /* Porównanie słów do n znaków */
    printf("\nstrncmp(\"prefix_rozne\", \"prefix_inne\", 7) = %d\n",
           strncmp("prefix_rozne", "prefix_inne", 7));  /* 0 – pierwsze 7 identyczne */

    /* ---- strcoll – porównanie z uwzględnieniem locale ---- */
    printf("\n=== strcoll vs strcmp ===\n\n");

    /* Bez locale (domyślnie C/POSIX): bazuje na ASCII */
    printf("Bez locale (C/POSIX):\n");
    printf("  strcmp(\"ą\", \"b\")   = %d\n", strcmp("ą", "b"));   /* > 0: kod UTF-8 ą > b */
    printf("  strcmp(\"z\", \"ą\")   = %d\n", strcmp("z", "ą"));   /* < 0 lub > 0 zależy od bajtu */

    /* Z polskim locale: ą powinno być między a i b */
    setlocale(LC_COLLATE, "pl_PL.UTF-8");  /* tylko dla sortowania */
    printf("\nZ polskim locale (pl_PL.UTF-8):\n");
    printf("  strcoll(\"a\", \"ą\")  = %d (a przed ą)\n",   strcoll("a", "ą"));   /* < 0 */
    printf("  strcoll(\"ą\", \"b\")  = %d (ą przed b)\n",   strcoll("ą", "b"));   /* < 0 */
    printf("  strcoll(\"z\", \"ź\")  = %d (z przed ź)\n",   strcoll("z", "ź"));   /* < 0 */

    /* ---- Praktyczny przykład: sortowanie tablicy stringów ---- */
    printf("\n=== Sortowanie z strcmp vs strcoll ===\n\n");

    const char *slowa[] = {"żaba", "ala", "ątrament", "bocian", "ćma"};
    int n = 5;

    /* Sortowanie bąbelkowe z strcmp */
    /* Tworzymy kopię tabeli wskaźników do sortowania */
    const char *kopia[5];
    for (int i = 0; i < n; i++) kopia[i] = slowa[i];

    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (strcmp(kopia[j], kopia[j+1]) > 0) {  /* strcmp, ASCII */
                const char *tmp = kopia[j];
                kopia[j] = kopia[j+1];
                kopia[j+1] = tmp;
            }
        }
    }
    printf("Sortowanie strcmp (ASCII):\n");
    for (int i = 0; i < n; i++) printf("  %s\n", kopia[i]);

    /* Sortowanie z strcoll */
    for (int i = 0; i < n; i++) kopia[i] = slowa[i];  /* reset */
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (strcoll(kopia[j], kopia[j+1]) > 0) {  /* strcoll, locale-aware */
                const char *tmp = kopia[j];
                kopia[j] = kopia[j+1];
                kopia[j+1] = tmp;
            }
        }
    }
    printf("Sortowanie strcoll (pl_PL.UTF-8):\n");
    for (int i = 0; i < n; i++) printf("  %s\n", kopia[i]);

    return 0;
}
```

## 12. `<string.h>` – wyszukiwanie

### Teoria

**`strchr(s, c)`** – szuka pierwszego wystąpienia znaku `c` w stringu `s`. Zwraca wskaźnik do znalezionego miejsca lub `NULL`.

**`strrchr(s, c)`** – jak `strchr`, ale szuka **ostatniego** wystąpienia (r = right/reverse).

**`strpbrk(s1, s2)`** – szuka pierwszego znaku z `s2` w `s1` (break = break at any of these chars). Przydatne gdy szukamy dowolnego ze zbioru znaków.

**`strstr(s1, s2)`** – szuka podstringa `s2` w `s1`. Zwraca wskaźnik do początku znalezionego podstringa lub `NULL`.

Wszystkie wyszukujące funkcje zwracają wskaźniki do wnętrza oryginalnego stringa – **nie kopiują** wynikowego tekstu.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strchr, strrchr, strpbrk, strstr */

int main(void) {
    /* ---- strchr – pierwsze wystąpienie znaku ---- */
    printf("=== strchr – pierwsze wystapienie ===\n\n");

    const char *tekst = "Ala ma kota i psa";
    printf("Tekst: \"%s\"\n\n", tekst);

    /* Szukamy pierwszego 'a' */
    char *znaleziony = strchr(tekst, 'a');
    if (znaleziony != NULL) {
        printf("Pierwsze 'a': na pozycji %td, reszta: \"%s\"\n",
               znaleziony - tekst,  /* różnica wskaźników = pozycja */
               znaleziony);
        /* wypisze: pozycja 2, reszta "a ma kota i psa" */
    }

    /* Szukamy 'z' – nie ma w stringu */
    char *brak = strchr(tekst, 'z');
    printf("Szukam 'z': %s\n", brak == NULL ? "nie znaleziono (NULL)" : "znaleziono");

    /* Szukamy wszystkich wystąpień 'a' */
    printf("\nWszystkie wystapienia 'a':\n");
    const char *poz = tekst;   /* zaczynamy od początku */
    while ((poz = strchr(poz, 'a')) != NULL) {  /* szukamy 'a' od poz */
        printf("  pozycja %td: \"%s\"\n", poz - tekst, poz);
        poz++;   /* przesuwamy się o 1 żeby nie utknąć w miejscu */
    }

    /* ---- strrchr – ostatnie wystąpienie ---- */
    printf("\n=== strrchr – ostatnie wystapienie ===\n\n");

    /* Typowe zastosowanie: znajdowanie rozszerzenia pliku */
    const char *sciezka = "/home/user/projekt/main.c";
    char *ostatnia_kropka = strrchr(sciezka, '.');
    if (ostatnia_kropka != NULL) {
        printf("Plik: \"%s\"\n", sciezka);
        printf("Rozszerzenie (od ostatniej kropki): \"%s\"\n", ostatnia_kropka);
        /* wypisze: ".c" */
        printf("Samo rozszerzenie (bez kropki): \"%s\"\n", ostatnia_kropka + 1);
        /* wypisze: "c" */
    }

    /* Znajdowanie nazwy pliku (ostatni '/') */
    char *ostatni_slash = strrchr(sciezka, '/');
    if (ostatni_slash != NULL) {
        printf("Nazwa pliku: \"%s\"\n", ostatni_slash + 1);  /* +1 pomija '/' */
        /* wypisze: "main.c" */
    }

    /* ---- strpbrk – pierwsza z grupy znaków ---- */
    printf("\n=== strpbrk – pierwszy znak ze zbioru ===\n\n");

    const char *zdanie = "To jest tekst: z przecinkami, kropkami.";
    printf("Zdanie: \"%s\"\n", zdanie);

    /* Szukamy pierwszego znaku interpunkcyjnego */
    char *interpunkcja = strpbrk(zdanie, ".,;:!?");
    if (interpunkcja != NULL) {
        printf("Pierwsza interpunkcja '%c' na pozycji %td\n",
               *interpunkcja, interpunkcja - zdanie);
        /* wypisze: ':' na pozycji 14 */
    }

    /* Iterowanie po tokenach rozdzielonych różnymi separatorami */
    printf("\nTokeny rozdzielone spacją, przecinkiem lub kropką:\n");
    const char *p = zdanie;
    while (*p != '\0') {
        /* Przeskakujemy separatory */
        p += strspn(p, " ,.");   /* strspn – ile znaków z podanego zestawu na początku */
        if (*p == '\0') break;

        /* Szukamy następnego separatora */
        char *sep = strpbrk(p, " ,.");
        if (sep == NULL) {
            printf("  \"%s\"\n", p);  /* ostatni token */
            break;
        }
        /* Wypisujemy fragment między pozycjami */
        printf("  \"");
        for (const char *q = p; q < sep; q++) putchar(*q);
        printf("\"\n");
        p = sep;  /* przesuwamy na separator */
    }

    /* ---- strstr – wyszukiwanie podstringa ---- */
    printf("\n=== strstr – wyszukiwanie podstringa ===\n\n");

    const char *haystack = "Programowanie w C jest fajne. C jest szybkie.";
    const char *needle = "C jest";

    printf("Tekst: \"%s\"\n", haystack);
    printf("Szukam: \"%s\"\n\n", needle);

    /* Pierwsze wystąpienie */
    char *znalezione = strstr(haystack, needle);
    if (znalezione != NULL) {
        printf("Pierwsze: pozycja %td, kontekst: \"%s\"\n",
               znalezione - haystack, znalezione);
    }

    /* Wszystkie wystąpienia */
    printf("\nWszystkie wystapienia \"%s\":\n", needle);
    const char *szukaj = haystack;
    int numer = 1;
    while ((szukaj = strstr(szukaj, needle)) != NULL) {
        printf("  %d. pozycja %td: \"%.20s...\"\n",
               numer++, szukaj - haystack, szukaj);
        szukaj += strlen(needle);  /* przeskakujemy znaleziony fragment */
    }

    /* Sprawdzanie czy string zawiera podstring */
    const char *czy_zawiera = "Programowanie";
    printf("\nCzy \"%s\" zawiera \"%s\": %s\n",
           haystack, czy_zawiera,
           strstr(haystack, czy_zawiera) != NULL ? "TAK" : "NIE");

    return 0;
}
```

## 13. `<string.h>` – strcspn i strspn

### Teoria

Te dwie funkcje zwracają **długość początkowego segmentu** spełniającego kryterium:

**`strcspn(s1, s2)`** – długość najdłuższego prefiksu `s1` złożonego ze znaków **NIE** należących do `s2`. (cspn = complement span)

**`strspn(s1, s2)`** – długość najdłuższego prefiksu `s1` złożonego ze znaków **należących** do `s2`. (spn = span)

Innymi słowy:
- `strcspn` – „ile znaków od początku zanim napotkamy znak z zakazanego zbioru"
- `strspn` – „ile znaków od początku należy do dozwolonego zbioru"

Zastosowania: pominięcie białych znaków, walidacja danych, tokenizacja.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strcspn, strspn */

int main(void) {
    /* ---- strcspn – długość segmentu BEZ znaków ze zbioru ---- */
    printf("=== strcspn ===\n\n");

    /* Jak długi jest prefix zanim pojawi się separator? */
    const char *csv = "imie,nazwisko,wiek";
    printf("String: \"%s\"\n", csv);

    size_t dlugosc_slowa = strcspn(csv, ",");  /* ile znaków do pierwszego ',' */
    printf("strcspn(\"%s\", \",\") = %zu\n", csv, dlugosc_slowa);
    /* wypisze: 4 – "imie" ma 4 znaki, potem jest przecinek */

    /* Wypisanie pierwszego tokenu */
    printf("Pierwszy token: \"");
    for (size_t i = 0; i < dlugosc_slowa; i++) putchar(csv[i]);
    printf("\"\n");

    /* Szukamy pierwszego znaku interpunkcyjnego lub cyfry */
    const char *tekst = "abcdef123!@#";
    size_t tylko_litery = strcspn(tekst, "0123456789!@#$%^&*");
    printf("\nPoczątkowe litery w \"%s\": %zu znaków\n", tekst, tylko_litery);
    /* wypisze: 6 – "abcdef" */

    /* Kiedy żaden znak nie pasuje – zwraca strlen */
    const char *czysty = "HelloWorld";
    size_t n1 = strcspn(czysty, "0123456789");  /* brak cyfr */
    printf("strcspn(\"%s\", cyfry) = %zu (= strlen = %zu)\n",
           czysty, n1, strlen(czysty));
    /* wypisze: 10 = strlen("HelloWorld") */

    /* ---- strspn – długość segmentu Z znaków ze zbioru ---- */
    printf("\n=== strspn ===\n\n");

    /* Ile znaków numerycznych na początku? */
    const char *numer = "12345abc";
    size_t ile_cyfr = strspn(numer, "0123456789");
    printf("strspn(\"%s\", cyfry) = %zu\n", numer, ile_cyfr);
    /* wypisze: 5 – "12345" to 5 cyfr, potem 'a' nie jest cyfrą */

    /* Pomijanie białych znaków na początku */
    const char *z_spacjami = "   \t  Witaj swiecie";
    size_t biale = strspn(z_spacjami, " \t\r\n");  /* spacja, tab, CR, LF */
    printf("Biale znaki na poczatku \"%s\": %zu\n", z_spacjami, biale);
    printf("Tekst po bialych: \"%s\"\n", z_spacjami + biale);
    /* wypisze: 6 białych znaków, "Witaj swiecie" */

    /* ---- Praktyczne zastosowanie: trim (usuwanie białych znaków) ---- */
    printf("\n=== Trim: usuwanie bialych znakow ===\n\n");

    /* Oryginalny string z białymi znakami */
    char do_trima[] = "   Hello World   ";
    printf("Przed trimem: \"%s\"\n", do_trima);

    /* ltrim – usuwanie z lewej strony (przesunięcie wskaźnika) */
    char *ltrimmed = do_trima + strspn(do_trima, " \t");
    printf("Po ltrim: \"%s\"\n", ltrimmed);

    /* rtrim – usuwanie z prawej strony */
    /* Szukamy ostatniego niebiałego znaku */
    char *koniec = ltrimmed + strlen(ltrimmed) - 1;  /* wskaźnik na ostatni znak */
    while (koniec > ltrimmed && (*koniec == ' ' || *koniec == '\t')) {
        koniec--;  /* cofamy się dopóki białe znaki */
    }
    *(koniec + 1) = '\0';  /* ucinamy białe znaki */
    printf("Po rtrim: \"%s\"\n", ltrimmed);

    /* ---- Walidacja: sprawdzanie czy string zawiera tylko dozwolone znaki ---- */
    printf("\n=== Walidacja danych ===\n\n");

    const char *dozwolone_w_nazie = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ_";
    const char *test1 = "mojaNazwa_zmiennej";
    const char *test2 = "zla-nazwa!";

    /* Jeśli strspn == strlen, to wszystkie znaki są dozwolone */
    size_t ok1 = strspn(test1, dozwolone_w_nazie);
    size_t ok2 = strspn(test2, dozwolone_w_nazie);

    printf("\"%s\": %s (strspn=%zu, strlen=%zu)\n",
           test1, ok1 == strlen(test1) ? "POPRAWNA" : "NIEPOPRAWNA",
           ok1, strlen(test1));
    printf("\"%s\": %s (strspn=%zu, strlen=%zu)\n",
           test2, ok2 == strlen(test2) ? "POPRAWNA" : "NIEPOPRAWNA",
           ok2, strlen(test2));

    /* ---- Tokenizacja ręczna z strcspn + strspn ---- */
    printf("\n=== Reczna tokenizacja ===\n\n");

    const char *wejscie = "  slowo1  slowo2   slowo3  ";
    const char *p = wejscie;
    printf("Tokeny z \"%s\":\n", wejscie);

    while (*p != '\0') {
        p += strspn(p, " \t");        /* pomiń białe znaki */
        if (*p == '\0') break;         /* koniec stringa */

        size_t dlugosc = strcspn(p, " \t");  /* długość tokenu */
        printf("  Token: \"");
        for (size_t i = 0; i < dlugosc; i++) putchar(p[i]);
        printf("\"\n");
        p += dlugosc;  /* przesuwamy za token */
    }

    return 0;
}
```

## 14. `<string.h>` – strlen i strtok

### Teoria

**`strlen(s)`** – zwraca liczbę znaków do `'\0'` (wyłącznie). Złożoność O(n) – musi przejść cały string.

**`strnlen_s(s, maxsize)`** (bezpieczna wersja) – nie przekracza `maxsize` bajtów.

**`strtok(s1, s2)`** – tokenizuje string `s1` rozdzielając wg znaków w `s2`:
- Pierwsze wywołanie: `strtok(s1, delimiters)` – zwraca pierwszy token
- Kolejne wywołania: `strtok(NULL, delimiters)` – zwraca kolejne tokeny
- **Modyfikuje oryginalny string!** Wstawia `'\0'` w miejsca separatorów.
- **Nie jest bezpieczna wielowątkowo** (używa globalnego stanu)
- Zwraca `NULL` gdy brak kolejnych tokenów

**`strtok_s`** (bezpieczna, wielowątkowa) – dodatkowy parametr `**ptr` przechowuje stan.

**Jak działa strtok:**  
String `"Ala,ma;kota"` po tokenizacji z delimitatorami `",;"`:
- Po 1. wywołaniu: `"Ala\0ma;kota"` → zwraca wskaźnik do `"Ala"`
- Po 2. wywołaniu: `"Ala\0ma\0kota"` → zwraca wskaźnik do `"ma"`
- Po 3. wywołaniu: zwraca wskaźnik do `"kota"`
- Po 4. wywołaniu: zwraca `NULL`

### Kod C

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strlen, strtok */
#include <stdlib.h>   /* malloc, free */

int main(void) {
    /* ---- strlen ---- */
    printf("=== strlen ===\n\n");

    /* Podstawowe użycie */
    const char *s1 = "Hello";
    const char *s2 = "";          /* pusty string */
    const char *s3 = "Ala\0ma";   /* string z osadzonym '\0' */

    printf("strlen(\"Hello\") = %zu\n",  strlen(s1));  /* 5 */
    printf("strlen(\"\") = %zu\n",        strlen(s2));  /* 0 */
    printf("strlen(\"Ala\\0ma\") = %zu\n", strlen(s3)); /* 3 – zatrzymuje się na '\0' */

    /* sizeof vs strlen */
    char tablica[] = "Hello";   /* tablica: Hello + '\0' = 6 bajtów */
    char *wskaznik = tablica;   /* wskaźnik: 8 bajtów (64-bit) lub 4 (32-bit) */

    printf("\nDla tablicy \"Hello\":\n");
    printf("  sizeof(tablica) = %zu  (cała tablica z '\\0')\n", sizeof(tablica));  /* 6 */
    printf("  strlen(tablica) = %zu  (długość bez '\\0')\n",   strlen(tablica));  /* 5 */
    printf("  sizeof(wskaznik) = %zu (rozmiar wskaźnika, nie stringa!)\n", sizeof(wskaznik)); /* 8 */
    printf("  strlen(wskaznik) = %zu (długość stringa, poprawnie)\n", strlen(wskaznik)); /* 5 */

    /* Obliczanie potrzebnego rozmiaru bufora */
    const char *tekst = "Programowanie";
    size_t potrzebny_bufor = strlen(tekst) + 1;  /* +1 na '\0' */
    printf("\nDla kopiowania \"%s\" potrzeba %zu bajtów\n", tekst, potrzebny_bufor);

    char *kopia = (char*)malloc(potrzebny_bufor);  /* dynamiczna alokacja */
    if (kopia != NULL) {
        memcpy(kopia, tekst, potrzebny_bufor);  /* kopiuj z '\0' */
        printf("Kopia: \"%s\"\n", kopia);
        free(kopia);   /* zwolnij pamięć */
    }

    /* ---- strtok ---- */
    printf("\n=== strtok ===\n\n");

    /* Podstawowy przykład z wykładu */
    char wejscie[] = "Ala,ma;kota i psa";  /* musimy zadeklarować jako tablicę, nie const! */
    const char *delimitatory = ",; ";        /* separatory: przecinek, średnik, spacja */

    printf("Wejście: \"%s\"\n", wejscie);
    printf("Separatory: \",; \"\n\n");

    /* Pierwsze wywołanie: przekazujemy string do podziału */
    char *token = strtok(wejscie, delimitatory);

    printf("Tokenizacja:\n");
    while (token != NULL) {
        printf("  Token: \"%s\"\n", token);
        token = strtok(NULL, delimitatory);  /* kolejne: NULL jako pierwszy argument */
    }
    /* wypisze: Ala, ma, kota, i, psa */

    /* Demonstracja modyfikacji oryginalnego stringa */
    printf("\nPo tokenizacji wejście zawiera '\\0' zamiast separatorów:\n");
    printf("Bajty: ");
    /* String "Ala,ma;kota i psa" po strtok wygląda jak "Ala\0ma\0kota\0i\0psa" */
    /* Wypisujemy bajty żeby to zobaczyć */
    for (int i = 0; i <= (int)strlen("Ala,ma;kota i psa"); i++) {
        printf("%02X ", (unsigned char)wejscie[i]);
    }
    printf("\n");

    /* ---- strtok z wieloma separatorami (ciągłe) ---- */
    printf("\n=== strtok z ciągłymi separatorami ===\n\n");

    char s_multi[] = "slowo1   slowo2\t\tslowo3";
    printf("Wejście: \"%s\"\n", s_multi);

    /* strtok ignoruje ciągłe separatory – nie zwraca pustych tokenów! */
    token = strtok(s_multi, " \t");
    while (token != NULL) {
        printf("  Token: \"%s\"\n", token);
        token = strtok(NULL, " \t");
    }

    /* ---- Zliczanie słów w zdaniu ---- */
    printf("\n=== Zliczanie slow ===\n\n");

    char zdanie[] = "Ala ma kota i psa i jeszcze jednego kota";
    printf("Zdanie: \"%s\"\n", zdanie);

    int liczba_slow = 0;
    char *slow = strtok(zdanie, " ");
    while (slow != NULL) {
        liczba_slow++;
        slow = strtok(NULL, " ");
    }
    printf("Liczba slow: %d\n", liczba_slow);  /* 9 */

    /* ---- Dlaczego strtok nie działa zagnieżdżony ---- */
    printf("\n=== strtok - brak zagniezdzen ===\n\n");

    char outer[] = "A:B:C";
    char inner[] = "1,2,3";

    printf("Outer: \"%s\", Inner: \"%s\"\n", outer, inner);
    printf("Problem: nie mozna uzywac strtok dla dwoch stringow naraz!\n");

    char *tok_a = strtok(outer, ":");  /* zaczyna tokenizację outer */
    /* tok_b = strtok(inner, ",");  // To ZEPSUJE tokenizację outer! */

    printf("Pierwsza iteracja outer z przerwa:\n");
    while (tok_a != NULL) {
        printf("  outer token: \"%s\"\n", tok_a);
        tok_a = strtok(NULL, ":");   /* kontynuacja outer */
    }

    return 0;
}
```

## 15. `<ctype.h>` – klasyfikacja i konwersja

### Teoria

`<ctype.h>` dostarcza funkcje sprawdzające typ znaku i konwertujące go. **Wszystkie przyjmują `int` (kod znaku) i zwracają `int` (niezerowe = prawda lub przekonwertowany znak).**

**Ważne:** argument musi być w zakresie `unsigned char` lub być wartością `EOF` (-1). Dla `signed char` o wartości ujemnej (np. -1) zachowanie jest **niezdefiniowane**. Dlatego zawsze rzutuj na `(unsigned char)`:

```c
isalpha((unsigned char)c)  /* bezpieczne */
```

**Funkcje klasyfikujące:**
| Funkcja | Sprawdza |
|---------|----------|
| `isalpha(c)` | litera (a-z, A-Z) |
| `isdigit(c)` | cyfra dziesiętna (0-9) |
| `isalnum(c)` | litera lub cyfra |
| `isupper(c)` | wielka litera |
| `islower(c)` | mała litera |
| `isspace(c)` | biały znak (spacja, tab, newline, itp.) |
| `isblank(c)` | spacja lub tab |
| `ispunct(c)` | znak interpunkcyjny |
| `isprint(c)` | znak drukowalny (ze spacją) |
| `isgraph(c)` | znak drukowalny (bez spacji) |
| `iscntrl(c)` | znak sterujący |
| `isxdigit(c)` | cyfra szesnastkowa (0-9, a-f, A-F) |

**Funkcje konwertujące:**
- `toupper(c)` – konwertuje na wielką literę (inne znaki bez zmian)
- `tolower(c)` – konwertuje na małą literę

**Uwaga:** `ctype.h` działa tylko z kodowaniem jednobajtowym. Dla Unicode używaj `<wctype.h>`.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <ctype.h>    /* isalpha, isdigit, toupper, itp. */

int main(void) {
    /* ---- Przegląd wszystkich funkcji klasyfikujących ---- */
    printf("=== Klasyfikacja znakow (ctype.h) ===\n\n");

    /* Testujemy różne znaki */
    int znaki[] = {'A', 'z', '5', ' ', '\t', '\n', '!', '#', '+', 'F', 127};
    const char *nazwy[] = {"'A'", "'z'", "'5'", "SP", "TAB", "LF",
                           "'!'", "'#'", "'+'", "'F'", "DEL"};
    int n = sizeof(znaki) / sizeof(znaki[0]);

    /* Nagłówek tabeli */
    printf("%-5s %4s %4s %4s %4s %4s %4s %4s %4s %4s %4s %4s %4s\n",
           "znak", "alph", "dgt", "alnu", "upr", "lwr", "spc", "blnk",
           "pnct", "prnt", "grph", "cntrl", "xdgt");
    printf("%s\n", "-------------------------------------------------------"
                   "-------------------------------");

    for (int i = 0; i < n; i++) {
        int c = znaki[i];
        /* Rzutowanie na unsigned char jest kluczowe! */
        printf("%-5s %4d %4d %4d %4d %4d %4d %4d %4d %4d %4d %4d %4d\n",
               nazwy[i],
               isalpha ((unsigned char)c) ? 1 : 0,
               isdigit ((unsigned char)c) ? 1 : 0,
               isalnum ((unsigned char)c) ? 1 : 0,
               isupper ((unsigned char)c) ? 1 : 0,
               islower ((unsigned char)c) ? 1 : 0,
               isspace ((unsigned char)c) ? 1 : 0,
               isblank ((unsigned char)c) ? 1 : 0,
               ispunct ((unsigned char)c) ? 1 : 0,
               isprint ((unsigned char)c) ? 1 : 0,
               isgraph ((unsigned char)c) ? 1 : 0,
               iscntrl ((unsigned char)c) ? 1 : 0,
               isxdigit((unsigned char)c) ? 1 : 0);
    }

    /* ---- toupper i tolower ---- */
    printf("\n=== toupper i tolower ===\n\n");

    const char *tekst = "Hello World 123!";
    printf("Oryginal:  \"%s\"\n", tekst);

    printf("toupper:   \"");
    for (int i = 0; tekst[i] != '\0'; i++) {
        putchar(toupper((unsigned char)tekst[i]));  /* konwersja na wielkie */
    }
    printf("\"\n");

    printf("tolower:   \"");
    for (int i = 0; tekst[i] != '\0'; i++) {
        putchar(tolower((unsigned char)tekst[i]));  /* konwersja na małe */
    }
    printf("\"\n");

    /* Ważna własność: toupper/tolower nie zmienia cyfr, znaków specjalnych */
    printf("\ntoupper('5') = '%c' (bez zmian)\n", toupper('5'));
    printf("toupper('!') = '%c' (bez zmian)\n", toupper('!'));
    printf("toupper('a') = '%c'\n", toupper('a'));  /* 'A' */
    printf("tolower('A') = '%c'\n", tolower('A'));  /* 'a' */

    /* ---- Praktyczne zastosowania ---- */
    printf("\n=== Praktyczne zastosowania ===\n\n");

    /* Walidacja liczby całkowitej */
    const char *napisy[] = {"12345", "-123", "12.34", "abc", "007", ""};
    for (int i = 0; napisy[i][0] != '\0' || i == 5; i++) {
        if (i == 5 && napisy[i][0] == '\0') {
            printf("\"\" -> pusty string\n");
            break;
        }
        const char *s = napisy[i];
        int j = 0;

        /* Opcjonalny znak minus */
        if (s[0] == '-' || s[0] == '+') j++;

        int jest_liczba = (s[j] != '\0');  /* nie może być puste po znaku */
        for (; s[j] != '\0'; j++) {
            if (!isdigit((unsigned char)s[j])) {
                jest_liczba = 0;
                break;
            }
        }
        printf("\"%-8s\" -> %s\n", s, jest_liczba ? "liczba calkowita" : "NIE jest liczbą");
    }

    /* Usunięcie wszystkich znaków niealfanumerycznych */
    printf("\n--- Usuwanie znaków niealfanumerycznych ---\n");
    const char *brudny = "H3ll0! W0rld, (co to jest?)";
    printf("Przed: \"%s\"\n", brudny);
    printf("Po:    \"");
    for (int i = 0; brudny[i] != '\0'; i++) {
        if (isalnum((unsigned char)brudny[i])) {  /* tylko alfanumeryczne */
            putchar(brudny[i]);
        }
    }
    printf("\"\n");

    /* Zliczanie różnych typów znaków w tekście */
    printf("\n--- Statystyki tekstu ---\n");
    const char *analiza = "Hello World! Mam 42 koty i 3 psy.";
    int litery = 0, cyfry = 0, spacje = 0, inne = 0;

    for (int i = 0; analiza[i] != '\0'; i++) {
        unsigned char c = (unsigned char)analiza[i];
        if (isalpha(c))       litery++;
        else if (isdigit(c))  cyfry++;
        else if (isblank(c))  spacje++;
        else                   inne++;
    }

    printf("Tekst: \"%s\"\n", analiza);
    printf("Litery: %d, Cyfry: %d, Spacje: %d, Inne: %d\n",
           litery, cyfry, spacje, inne);

    return 0;
}
```

## 16. `<wctype.h>` – znaki szerokie

### Teoria

`<wctype.h>` to odpowiednik `<ctype.h>` dla `wchar_t`. Funkcje działają na pełnych punktach kodowych Unicode (o ile `wchar_t` jest odpowiednio szeroki i locale jest poprawnie ustawione).

**Funkcje klasyfikujące** (odpowiedniki z `<ctype.h>`):
- `iswalpha(wc)` – litera Unicode (w tym ą, ę, ó itd.)
- `iswdigit(wc)` – cyfra
- `iswalnum(wc)` – litera lub cyfra
- `iswupper(wc)` – wielka litera
- `iswlower(wc)` – mała litera
- `iswspace(wc)` – biały znak
- `iswblank(wc)` – spacja lub tab
- `iswpunct(wc)` – interpunkcja
- `iswprint(wc)` – drukowalny
- `iswgraph(wc)` – drukowalny bez spacji
- `iswcntrl(wc)` – sterujący
- `iswxdigit(wc)` – cyfra szesnastkowa

**Funkcje konwertujące:**
- `towupper(wc)` – na wielką literę
- `towlower(wc)` – na małą literę
- `towctrans(wc, desc)` – ogólna transformacja

**Funkcje opisu transformacji:**
- `wctrans(name)` – zwraca deskryptor transformacji

### Kod C

```c
#include <stdio.h>     /* printf */
#include <wchar.h>     /* wchar_t, wcslen, printf %lc */
#include <wctype.h>    /* iswalpha, iswdigit, towupper, itp. */
#include <locale.h>    /* setlocale */

/* Wypisanie wchar_t stringa po znaku z klasyfikacją */
void analizuj_wstring(const wchar_t *s) {
    for (size_t i = 0; s[i] != L'\0'; i++) {
        wchar_t wc = s[i];
        printf("U+%04X '%lc': ", (unsigned)wc, wc);   /* %lc dla wchar_t */

        if (iswalpha(wc))  printf("litera ");
        if(iswupper(wc))   printf("WIELKA ");
        if(iswlower(wc))   printf("mała ");
        if(iswdigit(wc))   printf("cyfra ");
        if(iswalnum(wc))   printf("alnum ");
        if(iswspace(wc))   printf("biale ");
        if(iswpunct(wc))   printf("interpunkcja ");
        printf("\n");
    }
}

int main(void) {
    /* WAŻNE: locale musi być ustawione przed używaniem wctype funkcji */
    setlocale(LC_ALL, "pl_PL.UTF-8");

    /* ---- Klasyfikacja znaków wchar_t ---- */
    printf("=== isw* – klasyfikacja znakow szerokich ===\n\n");

    const wchar_t *test = L"Ala ąęó 123!";
    printf("Analiza: L\"%ls\"\n\n", test);
    analizuj_wstring(test);

    /* ---- towupper i towlower ---- */
    printf("\n=== towupper i towlower ===\n\n");

    const wchar_t *wtext = L"Cześć Świecie!";
    printf("Oryginal:  L\"%ls\"\n", wtext);

    printf("towupper:  L\"");
    for (size_t i = 0; wtext[i] != L'\0'; i++) {
        putwchar(towupper(wtext[i]));  /* putwchar – wchar_t na stdout */
    }
    printf("\"\n");

    printf("towlower:  L\"");
    for (size_t i = 0; wtext[i] != L'\0'; i++) {
        putwchar(towlower(wtext[i]));
    }
    printf("\"\n");

    /* ---- Porównanie ctype vs wctype ---- */
    printf("\n=== ctype.h vs wctype.h dla polskich liter ===\n\n");

    /* Polskie litery jako wchar_t */
    wchar_t polskie[] = L"ąćęłńóśźżĄĆĘŁŃÓŚŹŻ";
    printf("Polskie litery klasyfikacja iswalpha:\n");
    for (size_t i = 0; polskie[i] != L'\0'; i++) {
        printf("'%lc' (U+%04X): iswalpha=%d, iswupper=%d, iswlower=%d\n",
               polskie[i], (unsigned)polskie[i],
               iswalpha(polskie[i]),
               iswupper(polskie[i]),
               iswlower(polskie[i]));
    }

    /* ---- Zliczanie słów uwzględniając Unicode ---- */
    printf("\n=== Zliczanie slow z Unicode ===\n\n");

    const wchar_t *zdanie = L"Mam 3 koty, 2 psy i żółwia.";
    printf("Zdanie: L\"%ls\"\n", zdanie);

    int litery = 0, cyfry = 0, spacje = 0, inne = 0;
    for (size_t i = 0; zdanie[i] != L'\0'; i++) {
        wchar_t wc = zdanie[i];
        if (iswalpha(wc))      litery++;
        else if (iswdigit(wc)) cyfry++;
        else if (iswspace(wc)) spacje++;
        else                    inne++;
    }
    printf("Litery: %d (łącznie z polskimi!)\n", litery);
    printf("Cyfry:  %d\n", cyfry);
    printf("Spacje: %d\n", spacje);
    printf("Inne:   %d (przecinki, kropki itp.)\n", inne);

    /* ---- Konwersja wstring na wielkie litery ---- */
    printf("\n=== Konwersja na wielkie litery ===\n\n");

    wchar_t zdanie2[] = L"zażółć gęślą jaźń";
    printf("Przed: L\"%ls\"\n", zdanie2);
    for (size_t i = 0; zdanie2[i] != L'\0'; i++) {
        zdanie2[i] = (wchar_t)towupper(zdanie2[i]);  /* in-place konwersja */
    }
    printf("Po:    L\"%ls\"\n", zdanie2);
    /* Powinno wypisać: "ZAŻÓŁĆ GĘŚLĄ JAŹŃ" */

    return 0;
}
```

## 17. Zarządzanie pamięcią

### Teoria

Pamięć programu w C dzieli się na kilka obszarów:

**Stos (ang. *stack*):**
- Zmienne lokalne i parametry funkcji
- FILO (First In, Last Out) – jak stos talerzy
- Automatyczna alokacja i dealokacja (wchodzenie/wychodzenie z bloku)
- Ograniczony rozmiar (typowo 1–8 MB)
- Szybki dostęp, brak fragmentacji
- **Brak możliwości relokacji i zmiany rozmiaru**
- Kończąc blok, zmienne „znikają"

**Sterta (ang. *heap*):**
- Jawna alokacja (`malloc`, `calloc`, `realloc`) i dealokacja (`free`)
- Lub automatyczna przez Garbage Collector (nie dotyczy C)
- Duży rozmiar (ograniczony dostępną pamięcią)
- Możliwość relokacji (`realloc`) i zmiany rozmiaru
- **Wolniejszy dostęp, fragmentacja pamięci**
- Żyje przez cały czas działania programu (lub do `free`)

**Pamięć statyczna (ang. *static memory*):**
- Zmienne globalne i `static` zmienne lokalne
- Żyje przez cały czas działania programu
- Inicjalizacja do 0 przed startem `main()`

**Segment kodu/tekstu (ang. *code/text segment*):**
- Kod programu (skompilowane instrukcje)
- Literały stringów (`"Hello"`)
- **Tylko do odczytu** – modyfikacja powoduje segfault

### Kod C

```c
#include <stdio.h>     /* printf */
#include <stdlib.h>    /* malloc, calloc, realloc, free */
#include <string.h>    /* strlen, strcpy, memset */

/* Zmienna globalna – przechowywana w pamięci statycznej */
int globalna = 42;           /* jawna inicjalizacja */
int globalna_zero;           /* niejawna inicjalizacja do 0 */
char *statyczny_literal = "Hello";  /* wskaźnik w statycznej, literal w code segment */

/* Funkcja demonstrująca pamięć na stosie */
void pokaz_stos(void) {
    int lokalna = 10;        /* stos – żyje do końca funkcji */
    char bufor[64];           /* stos – 64 bajty zarezerwowane automatycznie */
    sprintf(bufor, "Wartosc: %d", lokalna);
    printf("Na stosie: %s\n", bufor);
    /* po powrocie z funkcji: lokalna i bufor są "zniszczone" */
}

/* Funkcja demonstrująca wyciek pamięci – BŁĄD */
void wyciek_pamieci(void) {
    char *s = (char*)malloc(100);  /* alokacja na stercie */
    strcpy(s, "Ten string wycieknie");
    printf("%s\n", s);
    /* brak free(s) – wyciek pamięci! */
}

/* Funkcja demonstrująca use-after-free – BŁĄD */
/* char* use_after_free(void) {       */
/*     char *s = (char*)malloc(10);   */
/*     free(s);                        */
/*     return s;  // BŁĄD: zwracamy wskaźnik do zwolnionej pamięci */
/* }                                   */

int main(void) {
    printf("=== Pamiec statyczna (global) ===\n\n");

    printf("globalna = %d (adres: %p)\n", globalna, (void*)&globalna);
    printf("globalna_zero = %d (inicjalizacja do 0)\n", globalna_zero);
    printf("literał \"Hello\" (read-only, adres: %p)\n", (void*)statyczny_literal);

    /* static lokalna – inicjalizowana raz, żyje przez cały program */
    {
        static int licznik = 0;   /* inicjalizowana do 0, tylko raz */
        licznik++;
        printf("static licznik = %d\n", licznik);
        /* za każdym wywołaniem bloku licznik rośnie, nie jest zerowany */
    }

    /* ---- Stos ---- */
    printf("\n=== Stos (stack) ===\n\n");

    pokaz_stos();   /* bufor i lokalna żyją tylko wewnątrz pokaz_stos */

    /* Zmienne na stosie */
    int x = 5;       /* int na stosie */
    char s[20];      /* tablica char na stosie */
    sprintf(s, "x = %d", x);
    printf("Stos: s = \"%s\" (adres: %p)\n", s, (void*)s);
    /* s jest dostępna dopóki jesteśmy w tym bloku */

    /* ---- Sterta ---- */
    printf("\n=== Sterta (heap) ===\n\n");

    /* malloc – alokacja bez inicjalizacji */
    int *ptr_int = (int*)malloc(sizeof(int));   /* 4 bajty na int */
    if (ptr_int == NULL) {  /* ZAWSZE sprawdzaj NULL */
        printf("Blad alokacji!\n");
        return 1;
    }
    *ptr_int = 99;   /* inicjalizujemy ręcznie */
    printf("malloc int: *ptr_int = %d (adres: %p)\n", *ptr_int, (void*)ptr_int);
    free(ptr_int);    /* OBOWIĄZKOWO zwalniamy */
    ptr_int = NULL;   /* dobra praktyka */

    /* calloc – alokacja z inicjalizacją do 0 */
    int *tablica = (int*)calloc(5, sizeof(int));  /* 5 intów, wszystkie = 0 */
    if (tablica == NULL) { printf("Blad calloc!\n"); return 1; }
    printf("calloc 5 intow: ");
    for (int i = 0; i < 5; i++) printf("%d ", tablica[i]);  /* wypisze: 0 0 0 0 0 */
    printf("\n");

    /* Używamy tablicy */
    for (int i = 0; i < 5; i++) tablica[i] = i * 10;
    printf("Po wypelnieniu: ");
    for (int i = 0; i < 5; i++) printf("%d ", tablica[i]);
    printf("\n");

    /* realloc – zmiana rozmiaru */
    int *wieksza = (int*)realloc(tablica, 10 * sizeof(int));  /* rozszerzamy do 10 */
    if (wieksza == NULL) {
        /* UWAGA: jeśli realloc zwróci NULL, stara pamięć NIE jest zwolniona! */
        free(tablica);  /* zwalniamy starą */
        printf("Blad realloc!\n");
        return 1;
    }
    tablica = wieksza;   /* aktualizujemy wskaźnik */
    /* Pierwsze 5 elementów zachowało wartości, nowe 5 są niezainicjalizowane */
    for (int i = 5; i < 10; i++) tablica[i] = i * 10;  /* inicjalizujemy nowe */

    printf("Po realloc do 10: ");
    for (int i = 0; i < 10; i++) printf("%d ", tablica[i]);
    printf("\n");
    free(tablica);  /* zwalniamy */
    tablica = NULL;

    /* ---- Dynamiczny string ---- */
    printf("\n=== Dynamiczny string na stercie ===\n\n");

    const char *src = "Dynamiczny tekst";
    size_t rozmiar = strlen(src) + 1;     /* +1 na '\0' */

    char *dyn_str = (char*)malloc(rozmiar);  /* alokacja dokładnie tyle ile trzeba */
    if (dyn_str == NULL) { printf("Blad!\n"); return 1; }

    strcpy(dyn_str, src);   /* kopiujemy tekst */
    printf("Dynamiczny string: \"%s\"\n", dyn_str);
    printf("Adres: %p (na stercie, nie na stosie)\n", (void*)dyn_str);

    /* Rozszerzenie dynamicznego stringa */
    const char *dodatek = " - rozszerzony";
    size_t nowy_rozmiar = strlen(dyn_str) + strlen(dodatek) + 1;
    char *rozszerzony = (char*)realloc(dyn_str, nowy_rozmiar);
    if (rozszerzony == NULL) { free(dyn_str); return 1; }
    dyn_str = rozszerzony;

    strcat(dyn_str, dodatek);   /* konkatenacja */
    printf("Po rozszerzeniu: \"%s\"\n", dyn_str);

    free(dyn_str);   /* koniecznie zwalniamy */
    dyn_str = NULL;

    /* ---- Typowe błędy pamięci ---- */
    printf("\n=== Demonstracja zasad ===\n\n");
    printf("Zasada 1: kazdy malloc/calloc/realloc wymaga free\n");
    printf("Zasada 2: po free ustawiaj wskaznik na NULL\n");
    printf("Zasada 3: nie uzywaj wskaznika po free (use-after-free)\n");
    printf("Zasada 4: nie free-uj dwa razy (double free)\n");
    printf("Zasada 5: nie zwracaj wskaznika do zmiennej lokalnej na stosie!\n");

    return 0;
}
```

**Błąd 2: Use-after-free**
```c
char *s = malloc(100);

```

**Błąd 3: Zwracanie wskaźnika do zmiennej lokalnej**

```c
char* zle(void) {

char* dobrze(void) {
    char *buf = malloc(100);  /* na stercie – żyje poza funkcją */

}
```

```c
char *s = malloc(10);

    free(s);  /* możemy zwolnić stary */
    return;
}
s = nowy;  /* teraz możemy zaktualizować */
```

## 18. Specyfikatory klas pamięci

### Teoria

Specyfikatory klas pamięci (`storage-class specifiers`) mówią kompilatorowi gdzie i jak długo ma żyć zmienna:

**`auto`** – domyślne dla zmiennych lokalnych, zmienne na stosie. Słowo kluczowe istnieje, ale jest archaizmem – nigdy nie piszemy go explicite w nowoczesnym C (w C++ `auto` ma inne znaczenie).

**`extern`** – zmienna jest zdefiniowana w **innym pliku** (lub niżej w tym samym). Nie alokuje pamięci – tylko deklaracja. Używane przy wieloplikowych projektach.

**`register`** – sugestia dla kompilatora: przechowuj tę zmienną w rejestrze CPU dla szybkości. Kompilator może ją zignorować. Nowoczesne kompilatory same optymalizują rejestry lepiej. `&register_var` jest **zabronione**.

**`static`** – dwa znaczenia:
- Dla zmiennej **lokalnej**: pamięć statyczna, inicjalizacja raz, wartość zachowywana między wywołaniami funkcji
- Dla zmiennej lub funkcji **globalnej**: ogranicza widoczność do bieżącego pliku (file scope)

**`thread_local`** (C11) – każdy wątek ma własną kopię zmiennej. Łączy się z `static` lub `extern`.

**`constexpr`** (C23) – wartość ustalana w czasie kompilacji, niemal jak `const`. Silniejsze gwarancje niż `const`.

### Kod C

```c
#include <stdio.h>    /* printf */
#include <threads.h>  /* thrd_t, thrd_create – C11 threads (opcjonalne) */

/* ---- extern – deklaracja zmiennej z innego pliku ---- */
/* W prawdziwym projekcie to byłoby w pliku nagłówkowym   */
/* Tutaj demonstrujemy tylko koncepcję                    */

/* Symulacja: deklarujemy że gdzieś indziej jest ta zmienna */
/* extern int zmienna_z_innego_pliku;  */

/* ---- zmienne globalne (pamięć statyczna) ---- */
int globalna_bez_static = 10;     /* widoczna w całym programie (i innych plikach) */
static int globalna_z_static = 20; /* widoczna tylko w tym pliku! */

/* ---- funkcja static – widoczna tylko w tym pliku ---- */
static void pomocnicza(void) {   /* static = plik-prywatna */
    printf("  Funkcja static: widoczna tylko w tym pliku\n");
}

/* Funkcja demonstrująca static lokalną */
int licznik_wywolan(void) {
    static int licznik = 0;  /* inicjalizacja TYLKO za pierwszym razem */
    licznik++;                /* zachowuje wartość między wywołaniami */
    return licznik;
}

/* Funkcja demonstrująca auto (domyślne) */
int suma_auto(int a, int b) {
    /* auto int wynik = a + b;  // archaizm, nigdy nie piszemy tego */
    int wynik = a + b;   /* to samo, domyślnie auto/stos */
    return wynik;
}

int main(void) {
    /* ---- auto (domyślna klasa) ---- */
    printf("=== auto (domyslna klasa pamieci) ===\n\n");

    int x = 5;              /* implicitly auto: stos, żyje do końca bloku */
    /* auto int y = 10;    // explicite auto: archaizm, nie rób tego */
    printf("int x = %d (domyslnie na stosie)\n", x);

    /* Blok wewnętrzny – nowy zakres */
    {
        int wewnetrzna = 100;   /* nowa zmienna, zasłania zewnętrzną jeśli ta sama nazwa */
        printf("Wewnatrz bloku: wewnetrzna = %d\n", wewnetrzna);
    }
    /* tutaj wewnetrzna już nie istnieje */

    /* ---- static lokalna ---- */
    printf("\n=== static (lokalna) ===\n\n");

    printf("Wywolania licznik_wywolan():\n");
    for (int i = 0; i < 5; i++) {
        int wartosc = licznik_wywolan();  /* static int licznik wewnątrz */
        printf("  Wywolanie %d: licznik = %d\n", i+1, wartosc);
    }
    /* wypisze: 1, 2, 3, 4, 5 – licznik rośnie, nie jest resetowany! */

    /* Porównanie: zmienna bez static jest resetowana */
    printf("\nPorownanie (bez static):\n");
    for (int i = 0; i < 3; i++) {
        int lokalna = 0;    /* reset przy każdym wywołaniu bloku */
        lokalna++;
        printf("  Iteracja %d: lokalna = %d (zawsze 1!)\n", i, lokalna);
    }

    /* ---- static globalna (file scope) ---- */
    printf("\n=== static (globalna) ===\n\n");

    printf("globalna_bez_static = %d (widoczna z innych plikow)\n",
           globalna_bez_static);
    printf("globalna_z_static = %d (widoczna TYLKO w tym pliku)\n",
           globalna_z_static);
    pomocnicza();   /* wywołanie static funkcji */

    /* ---- register ---- */
    printf("\n=== register ===\n\n");

    register int szybka = 42;   /* sugestia: przechowaj w rejestrze CPU */
    printf("register int szybka = %d\n", szybka);
    /* printf("adres: %p\n", &szybka);  // BŁĄD! nie można brać adresu register */

    /* Praktyczne użycie register: w pętlach (historycznie) */
    register int suma = 0;
    for (register int i = 0; i < 1000; i++) {  /* register na zmiennej pętli */
        suma += i;
    }
    printf("Suma 0-999 (z register): %d\n", suma);
    /* Nowoczesny kompilator z -O2 i tak to zoptymalizuje */

    /* ---- extern – demonstracja konceptu ---- */
    printf("\n=== extern ===\n\n");

    /* extern deklaruje, że zmienna istnieje gdzie indziej */
    /* W projekcie wieloplikowym: */
    /* plik1.c: int wspólna = 10;          // definicja */
    /* plik2.c: extern int wspólna;        // deklaracja */
    /* plik2.c: printf("%d", wspólna);     // użycie */

    printf("extern: zmienna z innego pliku – deklaracja, nie definicja\n");
    printf("Uzywane przy projektach wieloplikowych\n");

    /* ---- thread_local (C11) ---- */
    printf("\n=== thread_local ===\n\n");

    /* Każdy wątek ma własną kopię */
    _Thread_local int per_thread = 0;   /* C11: _Thread_local lub thread_local */
    per_thread = 42;
    printf("_Thread_local per_thread = %d (wlasciwosc biezacego watku)\n",
           per_thread);
    printf("W innych watkach ta zmienna ma odrebna wartosc\n");

    /* ---- constexpr (C23) ---- */
    printf("\n=== constexpr (C23) ===\n\n");

    /* constexpr – wartość ustalana w czasie kompilacji */
    /* constexpr int MAX = 100;      // C23 */
    /* Odpowiednik: #define MAX 100  // ale z typem */
    /* lub:         const int MAX = 100;  // const, ale może być runtime */

    const int MAX_CONST = 100;    /* const – może być ustalane runtime */
    /* constexpr int MAX_EXPR = 100; // C23: gwarantowane compile-time */

    printf("const int MAX_CONST = %d\n", MAX_CONST);
    printf("(C23) constexpr: wartosc ustalana w czasie kompilacji\n");
    printf("Silniejsza gwarancja niz const – zawsze compile-time\n");

    /* ---- Podsumowanie ---- */
    printf("\n=== Podsumowanie ===\n\n");
    printf("Klasa         | Gdzie      | Zyje jak dlugo      | Inicjalizacja\n");
    printf("auto (domysl) | stos       | do konca bloku      | niezdefin.\n");
    printf("static lokal  | statyczna  | przez caly program  | 0 (lub podana raz)\n");
    printf("static global | statyczna  | przez caly program  | 0 (lub podana)\n");
    printf("extern        | inna jedn. | przez caly program  | w def. pliku\n");
    printf("register      | (sugest.)  | do konca bloku      | niezdefin.\n");
    printf("thread_local  | per-watek  | przez zycie watku   | 0 (lub podana)\n");

    return 0;
}
```
