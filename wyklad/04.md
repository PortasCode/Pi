## 1. Operator adresu `&`

### Teoria
Każda zmienna, stała i funkcja zajmuje jakiś obszar pamięci RAM. Ten obszar ma swój adres — numer komórki pamięci, pod którym zaczyna się dany obiekt. Operator `&` (ampersand, „address-of") pozwala **odczytać ten adres**. Wynikiem wyrażenia `&zmienna` jest liczba całkowita (interpretowana jako adres), której typ zależy od typu zmiennej — np. dla `int a` wynikiem `&a` jest `int *` (wskaźnik na int).

Kilka ważnych obserwacji:
- `&a` daje adres zmiennej `a` — typ `int *`
- `&tekst` na tablicy znaków daje adres **całej tablicy** — typ `char (*)[50]`, nie `char *`
- `&suma` na funkcji daje jej adres w segmencie kodu — typ wskaźnika na funkcję
- Operatora `&` **nie można** stosować do wyrażeń tymczasowych (np. `&(a+b)` jest błędem)

### Kod

```c
#include <stdio.h>  /* biblioteka wejścia/wyjścia — potrzebna do printf */

/* definicja struktury używanej w przykładzie */
struct punkt3D {
    int x, y, z;
};

/* definicja funkcji — jej adres też można odczytać */
int suma(int a, int b) {
    return a + b;
}

int main(void) {
    int           a;                    /* zmienna całkowita */
    float         b;                    /* zmienna zmiennoprzecinkowa */
    struct punkt3D rog;                 /* zmienna strukturalna */
    char          tekst[50];            /* tablica znaków (łańcuch) */
    const double  y = 3.14;            /* stała zmiennoprzecinkowa */

    /* wypisanie adresów — %p to format dla wskaźnika/adresu */
    printf("Adres zmiennej a:      %p\n", (void *)&a);
    printf("Adres zmiennej b:      %p\n", (void *)&b);
    printf("Adres struktury rog:   %p\n", (void *)&rog);
    printf("Adres tablicy tekst:   %p\n", (void *)&tekst);
    printf("Adres stalej y:        %p\n", (void *)&y);
    printf("Adres funkcji suma:    %p\n", (void *)suma); /* dla funkcji: sama nazwa = adres */

    /* pokazanie, że adres tablicy i adres jej pierwszego elementu mają tę samą wartość */
    printf("\nAdres tablicy tekst:          %p\n", (void *)&tekst);
    printf("Adres 1. elementu (&tekst[0]): %p\n", (void *)&tekst[0]);
    printf("Sama nazwa tablicy (tekst):    %p\n", (void *)tekst);
    /* UWAGA: wartości są takie same, ale typy są różne!
       &tekst   -> char (*)[50]  — wskaźnik do całej tablicy
       tekst    -> char *        — wskaźnik do pierwszego elementu */

    return 0;
}
```

### Przykład wyjścia (adresy zależą od systemu)
```
Adres zmiennej a:      0x7ffd4a1b2c10
Adres zmiennej b:      0x7ffd4a1b2c14
Adres struktury rog:   0x7ffd4a1b2c18
Adres tablicy tekst:   0x7ffd4a1b2c24
Adres stalej y:        0x7ffd4a1b2c68
Adres funkcji suma:    0x5612ab3c1149

Adres tablicy tekst:           0x7ffd4a1b2c24
Adres 1. elementu (&tekst[0]): 0x7ffd4a1b2c24
Sama nazwa tablicy (tekst):    0x7ffd4a1b2c24
```
Widać, że `&tekst`, `&tekst[0]` i `tekst` mają **tę samą wartość liczbową**, ale niosą różną informację o typie.

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `printf("%p", &a)` bez rzutowania na `void *` | Formalnie UB w C — `%p` oczekuje `void *` | Zawsze rzutuj: `(void *)&a` |
| `&(a + b)` | Wyrażenie nie ma adresu (l-value) | Używaj `&` tylko na zmiennych |
| Mylenie `&tekst` z `tekst` | Mają tę samą wartość, ale różny typ — może spowodować błąd kompilacji przy przypisaniu | Rozróżniaj typy wskaźników |

## 2. Deklaracja wskaźnika

### Teoria
Wskaźnik to **zmienna przechowująca adres** innej zmiennej. Różni się od zwykłej zmiennej tym, że jej wartością jest adres, a nie bezpośrednia dana. Deklarując wskaźnik, podajemy typ danych, na który ma wskazywać — kompilator dzięki temu wie, ile bajtów odczytać podczas dereferencji i jak wykonywać arytmetykę wskaźnikową.

Schemat deklaracji:
```
typ *nazwa_wskaznika;
```

Gwiazdka `*` jest częścią deklaratora, **nie** nazwy zmiennej. Oznacza: „`nazwa_wskaznika` jest wskaźnikiem na `typ`".

Ważne przypadki:
- `int *wa` — wskaźnik na `int`
- `float *wb` — wskaźnik na `float`
- `struct punkt3D *wrog` — wskaźnik na strukturę
- `int **wwa` — wskaźnik na wskaźnik na `int` (podwójne pośrednictwo)
- `int (*wsuma)(int, int)` — wskaźnik na funkcję przyjmującą dwa `int`-y i zwracającą `int` (nawiasy wokół `*wsuma` są konieczne!)

### Kod

```c
#include <stdio.h>    /* printf */
#include <stddef.h>   /* NULL, size_t */

/* struktura pomocnicza */
struct punkt3D {
    int x, y, z;
};

/* funkcja, której adres zapiszemy w wskaźniku na funkcję */
int dodaj(int a, int b) {
    return a + b;
}

int main(void) {
    /* --- deklaracje wskaźników różnych typów --- */

    int           a   = 42;            /* zwykła zmienna całkowita */
    float         b   = 3.14f;         /* zwykła zmienna float */
    struct punkt3D rog = {1, 2, 3};    /* zmienna strukturalna */

    int           *wa;                 /* wskaźnik na int — niezainicjowany! */
    float         *wb;                 /* wskaźnik na float — niezainicjowany! */
    struct punkt3D *wrog;              /* wskaźnik na struct punkt3D — niezainicjowany! */
    int           **wwa;               /* wskaźnik na wskaźnik na int */
    int           (*wdodaj)(int, int); /* wskaźnik na funkcję int(int,int) */

    /* --- inicjalizacja wskaźników adresami zmiennych --- */

    wa    = &a;                        /* wa wskazuje na zmienną a */
    wb    = &b;                        /* wb wskazuje na zmienną b */
    wrog  = &rog;                      /* wrog wskazuje na strukturę rog */
    wwa   = &wa;                       /* wwa wskazuje na wskaźnik wa */
    wdodaj = dodaj;                    /* nazwa funkcji = jej adres (& opcjonalne) */

    /* --- wypisanie rozmiarów wskaźników --- */
    /* na platformach 64-bitowych wszystkie wskaźniki mają 8 bajtów */
    printf("sizeof(int *)             = %zu bajtow\n", sizeof(int *));
    printf("sizeof(float *)           = %zu bajtow\n", sizeof(float *));
    printf("sizeof(struct punkt3D *)  = %zu bajtow\n", sizeof(struct punkt3D *));
    printf("sizeof(int **)            = %zu bajtow\n", sizeof(int **));
    printf("sizeof wskaznika na funk. = %zu bajtow\n", sizeof(int (*)(int, int)));

    /* --- demonstracja podwójnego pośrednictwa --- */
    printf("\nWartość a przez wa:   %d\n", *wa);     /* dereferencja 1 poziom */
    printf("Wartość a przez wwa:  %d\n", **wwa);    /* dereferencja 2 poziomy */

    /* --- wywołanie funkcji przez wskaźnik --- */
    int wynik = wdodaj(10, 20);        /* wywołanie przez wskaźnik jak zwykła funkcja */
    printf("dodaj(10,20) przez wskaznik = %d\n", wynik);

    return 0;
}
```

### Przykład wyjścia
```
sizeof(int *)             = 8 bajtow
sizeof(float *)           = 8 bajtow
sizeof(struct punkt3D *)  = 8 bajtow
sizeof(int **)            = 8 bajtow
sizeof wskaznika na funk. = 8 bajtow

Wartość a przez wa:   42
Wartość a przez wwa:  42
dodaj(10,20) przez wskaznik = 30
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `int* wa, wb;` | Deklaruje `int *wa` i `int wb` (nie dwa wskaźniki!) | Pisz `int *wa, *wb;` lub osobno |
| Używanie niezainicjowanego wskaźnika | Losowy adres → crash lub UB | Zawsze inicjuj: `int *wa = NULL;` lub `int *wa = &a;` |
| `int *wdodaj = dodaj;` zamiast `int (*wdodaj)(int,int)` | Zły typ — kompilator może odrzucić lub zachowanie niezdefiniowane | Zawsze podaj pełną sygnaturę wskaźnika na funkcję |

## 3. Wskaźnik pusty (`nullptr` / `NULL`)

### Teoria
Wskaźnik pusty to wskaźnik, który **nie wskazuje na żadną zmienną**. Jest to bezpieczna wartość „brak wskazania". W C używamy:
- `NULL` — makro z `<stddef.h>` lub `<stdlib.h>`, historycznie `((void*)0)` lub `0`
- `0` — literał całkowity zerowywany do wskaźnika w kontekście wskaźnikowym
- `nullptr` — słowo kluczowe wprowadzone w **C23**, jednoznaczny pusty wskaźnik (bezpieczniejszy, bo ma własny typ `nullptr_t`)

**Nigdy nie wolno dereferencjonować pustego wskaźnika** — powoduje to błąd segmentacji (Segmentation Fault). Przed każdą dereferencją należy sprawdzić, czy wskaźnik nie jest pusty.

Wzorzec sprawdzania:
```c
if (wsk != NULL) { /* wskaźnik jest ważny */ }
if (wsk == NULL) { /* wskaźnik jest pusty */ }
// skrócona wersja (działa bo NULL == 0, co jest fałszem):
if (wsk)   { /* ważny */ }
if (!wsk)  { /* pusty */ }
```

### Kod

```c
#include <stdio.h>    /* printf */
#include <stdlib.h>   /* NULL, malloc, free */

/* funkcja bezpieczna — zawsze sprawdza wskaźnik przed użyciem */
void wypisz_wartosc(int *wsk) {
    if (wsk == NULL) {                          /* sprawdzenie przed dereferencją */
        printf("Wskaznik jest pusty (NULL)!\n");
        return;                                  /* wyjście z funkcji bez awarii */
    }
    printf("Wartość pod wskaźnikiem: %d\n", *wsk); /* dereferencja tylko gdy bezpiecznie */
}

int main(void) {
    int  a = 100;                /* zwykła zmienna */
    int *wa = NULL;              /* wskaźnik zainicjowany jako pusty — dobra praktyka */

    /* sprawdzenie przed użyciem */
    printf("--- Test 1: wskaznik pusty ---\n");
    wypisz_wartosc(wa);          /* wa == NULL — funkcja wypisze komunikat */

    /* przypisanie poprawnego adresu */
    wa = &a;

    printf("\n--- Test 2: wskaznik wazny ---\n");
    wypisz_wartosc(wa);          /* wa != NULL — funkcja wypisze 100 */

    /* zerowanie wskaźnika po użyciu — zapobiega "dangling pointer" */
    wa = NULL;
    printf("\n--- Test 3: wskaznik wyzerowany po uzyciu ---\n");
    wypisz_wartosc(wa);

    /* typowy wzorzec: if (wsk) zamiast if (wsk != NULL) */
    int b = 55;
    int *wb = &b;
    if (wb) {                    /* skrócony zapis — oba zapisy są równoważne */
        printf("\nwb wskazuje na: %d\n", *wb);
    }

    /* porównanie NULL, 0 i nullptr (nullptr wymaga C23, tu używamy NULL) */
    int *wc = NULL;              /* NULL — klasyczny sposób */
    int *wd = 0;                 /* 0 — dopuszczalne, ale mniej czytelne */
    if (wc == wd) {
        printf("NULL i 0 jako wskaznik sa rowne\n");
    }

    return 0;
}
```

### Przykład wyjścia
```
--- Test 1: wskaznik pusty ---
Wskaznik jest pusty (NULL)!

--- Test 2: wskaznik wazny ---
Wartość pod wskaźnikiem: 100

--- Test 3: wskaznik wyzerowany po uzyciu ---
Wskaznik jest pusty (NULL)!

wb wskazuje na: 55
NULL i 0 jako wskaznik sa rowne
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| Dereferencja bez sprawdzenia | Jeśli wsk == NULL → Segmentation Fault | Zawsze sprawdzaj przed `*wsk` |
| Niezainicjowany wskaźnik zamiast NULL | Wskaźnik ma losowy śmieciowy adres — może „działać" przypadkowo | Inicjuj każdy wskaźnik: `int *p = NULL;` |
| Brak zerowania po `free()` | Dangling pointer — wskaźnik nadal trzyma stary adres | Po `free(p)` zawsze pisz `p = NULL;` |

## 4. Wskaźnik bez typu (`void *`)

### Teoria
`void *` to wskaźnik **bez określonego typu** — „surowy adres". Może przechować adres zmiennej dowolnego typu. Jest używany tam, gdzie typ danych nie jest z góry znany, np. w funkcjach bibliotecznych takich jak `malloc()`, `memcpy()`, `qsort()`.

Ograniczenia `void *`:
- **Nie można dereferencjonować** bezpośrednio (kompilator nie wie, ile bajtów odczytać)
- **Nie można wykonywać arytmetyki** wskaźnikowej (nie wiadomo, ile bajtów ma jeden krok)
- Przed użyciem trzeba **rzutować** na konkretny typ: `(int *) wsk`

W C (w odróżnieniu od C++) wskaźnik `void *` można przypisywać do wskaźnika dowolnego typu i odwrotnie **bez jawnego rzutowania**, ale rzutowanie jest dobrą praktyką zwiększającą czytelność.

### Kod

```c
#include <stdio.h>    /* printf */
#include <stdlib.h>   /* malloc, free */

int main(void) {
    int    x = 132;              /* zmienna całkowita */
    float  f = 3.14f;           /* zmienna zmiennoprzecinkowa */
    double d = 2.718;           /* zmienna podwójnej precyzji */

    void *wsk;                   /* wskaźnik bez typu — nie wskazuje jeszcze na nic */

    /* przypisanie adresu int do void* */
    wsk = &x;                    /* w C: można bez rzutowania; void* przyjmuje każdy adres */
    printf("Adres x przez void*: %p\n", wsk);

    /* żeby odczytać wartość, MUSIMY rzutować na właściwy typ */
    int *wa = (int *) wsk;       /* rzutowanie void* na int* */
    printf("Wartość x przez rzutowanie: %d\n", *wa);

    /* można też rzutować w miejscu bez pośredniej zmiennej */
    printf("Wartość x inline cast: %d\n", *((int *) wsk));

    /* zmiana wskazania na float */
    wsk = &f;                    /* teraz wsk przechowuje adres zmiennej f */
    float *wf = (float *) wsk;   /* rzutowanie void* na float* */
    printf("Wartość f przez void*: %.2f\n", *wf);

    /* zmiana wskazania na double */
    wsk = &d;
    printf("Wartość d przez void*: %.3f\n", *((double *) wsk));

    /* --- praktyczny przykład: malloc zwraca void* --- */
    /* malloc alokuje pamięć i zwraca void* — rzutujemy na potrzebny typ */
    int *tablica = (int *) malloc(5 * sizeof(int));  /* alokacja tablicy 5 intów */

    if (tablica == NULL) {        /* zawsze sprawdzamy czy malloc się powiódł */
        printf("Blad alokacji pamieci!\n");
        return 1;
    }

    /* wypełnienie i wypisanie tablicy */
    for (int i = 0; i < 5; i++) {
        tablica[i] = (i + 1) * 10;  /* wartości: 10, 20, 30, 40, 50 */
    }
    printf("\nTablica dynamiczna: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", tablica[i]);
    }
    printf("\n");

    free(tablica);               /* zwolnienie pamięci — obowiązkowe! */
    tablica = NULL;              /* zerowanie wskaźnika po free */

    return 0;
}
```

### Przykład wyjścia
```
Adres x przez void*: 0x7ffd3c2a1b04
Wartość x przez rzutowanie: 132
Wartość x inline cast: 132
Wartość f przez void*: 3.14
Wartość d przez void*: 2.718

Tablica dynamiczna: 10 20 30 40 50
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `*wsk` gdzie `wsk` to `void *` | Błąd kompilacji — kompilator nie zna rozmiaru | Zawsze rzutuj przed dereferencją: `*(int *)wsk` |
| Rzutowanie na zły typ | Odczyt niepoprawnej liczby bajtów — UB | Pamiętaj, jaki typ był oryginalnie zapisany |
| Pominięcie sprawdzenia `NULL` po `malloc` | `malloc` może zwrócić NULL przy braku pamięci | Zawsze `if (ptr == NULL) { ... }` po malloc |

## 5. Wskaźnik do stałej vs. stały wskaźnik

### Teoria
Kombinacja `const` i `*` daje trzy różne przypadki. Kluczowe pytanie: **co jest stałe — wartość pod wskaźnikiem czy sam wskaźnik?**

**Reguła czytania od prawej do lewej** (ang. *right-left rule*):
- `const int *wx` → czytaj: `wx` jest wskaźnikiem (`*`) na `int` który jest `const` → **wskaźnik do stałej** (wartości nie możemy zmienić przez wskaźnik, ale sam wskaźnik możemy przestawić)
- `int * const wy` → czytaj: `wy` jest `const` wskaźnikiem (`*`) na `int` → **stały wskaźnik** (wskaźnik zawsze wskazuje to samo miejsce, ale wartość tam możemy zmieniać)
- `const int * const wz` → oboje `const` → **stały wskaźnik do stałej** (ani wartość, ani adres nie mogą być zmienione)

Typowe zastosowania:
- `const typ *` — argumenty funkcji, które nie powinny modyfikować danych (read-only parameter)
- `typ * const` — np. `this` w C++, wskaźniki bazowe tablic
- `const typ * const` — absolutnie niezmienialny uchwyt

### Kod

```c
#include <stdio.h>    /* printf */

int main(void) {
    const int x = 5;             /* stała całkowita — nie można zmienić jej wartości */
    int       y = 20;            /* zwykła zmienna */
    int       z = 99;            /* inna zwykła zmienna */

    /* ===== PRZYPADEK 1: wskaźnik do stałej ===== */
    /* const int *wx  — "wx jest wskaźnikiem na const int" */
    /* Nie można zmienić *wx (wartości), ale można zmienić wx (adres) */
    const int *wx = &x;          /* wx wskazuje na stałą x */

    printf("--- Wskaznik do stalej (const int *wx) ---\n");
    printf("*wx = %d\n", *wx);   /* odczyt OK */
    /* *wx = 10; */              /* BŁĄD KOMPILACJI: nie można modyfikować przez wskaźnik do stałej */

    wx = &y;                     /* OK: można przestawić sam wskaźnik na inny adres */
    printf("Po przestawieniu wx na y: *wx = %d\n", *wx);

    /* ===== PRZYPADEK 2: stały wskaźnik do zmiennej ===== */
    /* int * const wy  — "wy jest const wskaźnikiem na int" */
    /* Można zmienić *wy (wartość), ale nie można zmienić wy (adresu) */
    int * const wy = &y;         /* wy ZAWSZE wskazuje na y — musi być zainicjowany od razu */

    printf("\n--- Staly wskaznik do zmiennej (int * const wy) ---\n");
    printf("Przed: *wy = %d\n", *wy);
    *wy = 55;                    /* OK: modyfikacja wartości pod stałym wskaźnikiem */
    printf("Po *wy=55: y = %d\n", y);
    /* wy = &z; */               /* BŁĄD KOMPILACJI: nie można zmienić adresu stałego wskaźnika */

    /* ===== PRZYPADEK 3: stały wskaźnik do stałej ===== */
    /* const int * const wz  — "wz jest const wskaźnikiem na const int" */
    /* Nie można zmieniać ani wartości, ani adresu */
    const int * const wz = &x;  /* musi być zainicjowany od razu */

    printf("\n--- Staly wskaznik do stalej (const int * const wz) ---\n");
    printf("*wz = %d\n", *wz);   /* odczyt OK */
    /* *wz = 10; */              /* BŁĄD KOMPILACJI */
    /* wz = &y;  */              /* BŁĄD KOMPILACJI */

    /* ===== Praktyczne zastosowanie: funkcja tylko do odczytu ===== */
    /* Parametr (const int *tab) oznacza: "obiecuję nie modyfikować tablicy" */
    printf("\n--- Demonstracja const w argumencie funkcji ---\n");
    const int dane[3] = {10, 20, 30};
    const int *ptr = dane;       /* wskaźnik do stałej tablicy */
    for (int i = 0; i < 3; i++) {
        printf("dane[%d] = %d\n", i, ptr[i]);  /* odczyt przez wskaźnik — OK */
    }

    return 0;
}
```

### Przykład wyjścia
```
--- Wskaznik do stalej (const int *wx) ---
*wx = 5
Po przestawieniu wx na y: *wx = 20

--- Staly wskaznik do zmiennej (int * const wy) ---
Przed: *wy = 20
Po *wy=55: y = 55

--- Staly wskaznik do stalej (const int * const wz) ---
*wz = 5

--- Demonstracja const w argumencie funkcji ---
dane[0] = 10
dane[1] = 20
dane[2] = 30
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| Mylenie `const int *p` z `int * const p` | Różne ograniczenia — błąd logiczny | Czytaj od prawej: co jest `const`? |
| `int * const p;` bez inicjalizacji | Stały wskaźnik musi być zainicjowany w deklaracji | Zawsze: `int * const p = &x;` |
| Rzutowanie `const` na nie-`const` (odrzucanie const) | Modyfikacja stałej → UB, ostrzeżenie kompilatora | Nigdy nie odrzucaj `const` rzutowaniem |

## 6. Dereferencja — operator `*`

### Teoria
Dereferencja (wyłuskanie) to **odczytanie lub zapis wartości pod adresem przechowywanym przez wskaźnik**. Operator dereferencji `*` zastosowany przed wskaźnikiem daje dostęp do zmiennej, na którą wskaźnik wskazuje.

```
wa  — adres zmiennej a (np. 0x7ffd...)
*wa — wartość zmiennej a (np. 42)
```

Operacji dereferencji **nie można** wykonać na:
- wskaźniku pustym (`NULL`) — Segmentation Fault
- wskaźniku `void *` — błąd kompilacji (nieznany rozmiar)
- niezainicjowanym wskaźniku — undefined behaviour

Priorytety operatorów przy `*` i `++`:
- `*wa++` → `*(wa++)` — zwiększ wskaźnik (nie wartość!), potem dereferencjonuj
- `(*wa)++` → zwiększ wartość pod wskaźnikiem
- `++*wa` → `++(* wa)` — zwiększ wartość pod wskaźnikiem (pre-increment)
- `*++wa` → `*(++wa)` — przesuń wskaźnik do przodu, potem dereferencjonuj

### Kod

```c
#include <stdio.h>    /* printf */

int main(void) {
    int a = 10;                  /* zmienna, na którą będziemy wskazywać */
    int *wa = &a;                /* wa przechowuje adres zmiennej a */

    /* --- podstawowy odczyt i zapis przez dereferencję --- */
    printf("a = %d\n", a);       /* bezpośredni dostęp */
    printf("*wa = %d\n", *wa);   /* dereferencja — to samo co a */

    *wa = 5;                     /* zapis przez wskaźnik — zmienia wartość a! */
    printf("Po *wa=5: a = %d\n", a);  /* a jest teraz 5 */

    /* --- dereferencja w wyrażeniu --- */
    int suma = 0;
    suma = suma + *wa;           /* dodanie wartości spod wskaźnika do sumy */
    printf("suma = 0 + *wa(%d) = %d\n", *wa, suma);

    /* --- różnica między (*wa)++ a *wa++ --- */
    int tab[5] = {100, 200, 300, 400, 500};  /* tablica do demonstracji */
    int *wt = tab;               /* wt wskazuje na tab[0] */

    printf("\n--- Demonstracja priorytetow ---\n");
    printf("Poczatkowy: *wt = %d (tab[0])\n", *wt);

    /* (*wa)++ — zwiększa WARTOŚĆ pod wskaźnikiem, wskaźnik nie rusza się */
    int b = 50;
    int *wb = &b;
    printf("\nb przed (*wb)++: %d\n", b);
    (*wb)++;                     /* zwiększ b (wartość pod wb) o 1 */
    printf("b po  (*wb)++: %d\n", b);    /* b = 51, wb nadal wskazuje na b */

    /* *wt++ — dereferencjonuje BIEŻĄCĄ pozycję, potem przesuwa wskaźnik */
    printf("\nIteracja przez tablicę używając *wt++:\n");
    wt = tab;                    /* reset wskaźnika na początek tablicy */
    for (int i = 0; i < 5; i++) {
        printf("  element: %d\n", *wt++);  /* odczytuje *wt, potem wt++ */
    }
    /* po pętli wt wskazuje ZA tablicą — nie dereferencjonuj! */

    /* *++wt — najpierw przesuwa wskaźnik, potem dereferencjonuje */
    printf("\nIteracja używając *++wt:\n");
    wt = tab;                    /* reset na tab[0] */
    printf("  Startujemy od tab[0] = %d\n", *wt);
    for (int i = 0; i < 4; i++) {
        printf("  nastepny: %d\n", *++wt);  /* wt++, potem odczyt */
    }

    return 0;
}
```

### Przykład wyjścia
```

*wa = 10
Po *wa=5: a = 5
suma = 0 + *wa(5) = 5

--- Demonstracja priorytetow ---
Poczatkowy: *wt = 100 (tab[0])

b przed (*wb)++: 50
b po  (*wb)++: 51

Iteracja przez tablicę używając *wt++:
  element: 100
  element: 200
  element: 300
  element: 400
  element: 500

Iteracja używając *++wt:
  Startujemy od tab[0] = 100
  nastepny: 200
  nastepny: 300
  nastepny: 400
  nastepny: 500
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `*wa++` zamiast `(*wa)++` | Przesuwa wskaźnik zamiast inkrementować wartość | Używaj nawiasów: `(*wa)++` |
| Dereferencja NULL | Segmentation Fault | Sprawdź: `if (wa != NULL)` |
| Dereferencja po zakończeniu pętli `*wt++` | `wt` wskazuje za tablicę | Nie dereferencjonuj wskaźnika po zakończeniu iteracji |

## 7. Wskaźnik do elementu tablicy

### Teoria
W C **nazwa tablicy jest automatycznie konwertowana** (ang. *array decay*) na wskaźnik do jej pierwszego elementu w większości kontekstów (wyjątki: operator `sizeof`, `&tablica`). Oznacza to:

```c
int tab[5] = {7, 5, 2, 1, 8};
int *w = tab;       // to samo co: int *w = &tab[0];
```

Między `tab` (wskaźnik na pierwszy element) a `&tab` (wskaźnik na całą tablicę) jest subtelna różnica typów:
- `tab` → typ `int *` — wskaźnik na `int`
- `&tab` → typ `int (*)[5]` — wskaźnik na tablicę 5 intów

Obie wartości liczbowe są równe, ale arytmetyka wskaźnikowa da różne wyniki (`tab+1` to kolejny int, `&tab+1` to adres za całą tablicą).

Dostęp do elementów przez wskaźnik:
```c
w[i]      // notacja tablicowa — najczytelniejsza
*(w + i)  // notacja wskaźnikowa — równoważna
```

### Kod

```c
#include <stdio.h>    /* printf */

int main(void) {
    int tab[5] = {7, 5, 2, 1, 8};   /* tablica 5 elementów */

    /* === Sposób 1: wskaźnik na pierwszy element === */
    int *w = tab;                    /* to samo co: int *w = &tab[0]; */

    printf("=== Wskaznik na element (int *w) ===\n");
    printf("Adres tab:      %p\n", (void *)tab);
    printf("Adres &tab[0]:  %p\n", (void *)&tab[0]);
    printf("Adres w:        %p\n", (void *)w);
    /* trzy powyższe powinny być identyczne */

    /* dostęp przez wskaźnik — dwie równoważne notacje */
    printf("\nElementy przez notacje wskaznikowa *(w+i):\n");
    for (int i = 0; i < 5; i++) {
        printf("  *(w+%d) = %d\n", i, *(w + i));
    }

    printf("\nElementy przez notacje tablicowa w[i]:\n");
    for (int i = 0; i < 5; i++) {
        printf("  w[%d] = %d\n", i, w[i]);    /* równoważne *(w+i) */
    }

    /* === Sposób 2: wskaźnik na całą tablicę === */
    int (*wt)[5] = &tab;             /* wskaźnik na tablicę 5 intów */

    printf("\n=== Wskaznik na tablice (int (*wt)[5]) ===\n");
    printf("Adres &tab:  %p\n", (void *)&tab);
    printf("Adres wt:    %p\n", (void *)wt);
    /* wartości identyczne z powyższymi, ale typ inny! */

    /* dereferencja wt daje tablicę, double-dereferencja daje element */
    printf("Dostep do elementu: (*wt)[2] = %d\n", (*wt)[2]);  /* tab[2] = 2 */

    /* === Różnica w arytmetyce wskaźnikowej === */
    printf("\n=== Arytmetyka: roznica miedzy w i &tab ===\n");
    printf("w      = %p\n", (void *)w);
    printf("w + 1  = %p  (przeskoczyl o %zu bajtow = sizeof(int))\n",
           (void *)(w + 1), (size_t)((char*)(w+1) - (char*)w));
    printf("&tab   = %p\n", (void *)&tab);
    printf("&tab+1 = %p  (przeskoczyl o %zu bajtow = sizeof(int[5]))\n",
           (void *)(&tab + 1), (size_t)((char*)(&tab+1) - (char*)&tab));

    return 0;
}
```

### Przykład wyjścia

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `int *w = &tab` zamiast `int *w = tab` | `&tab` to `int(*)[5]`, nie `int *` — błąd typów | Używaj `tab` lub `&tab[0]` dla wskaźnika na element |
| Wyjście poza tablicę przez wskaźnik | UB — możliwy crash lub błąd logiczny | Pilnuj granic: `w + i` gdzie `0 <= i < rozmiar` |
| `sizeof(w)` zamiast `sizeof(tab)` | Wskaźnik to 8 bajtów, nie rozmiar tablicy | Przekazuj rozmiar tablicy osobnym parametrem |

## 8. Arytmetyka wskaźników

### Teoria
Na wskaźnikach można wykonywać **ograniczoną arytmetykę**:
- `wsk + n` → przesuń o `n` elementów do przodu (nie bajtów! — o `n * sizeof(typ)`)
- `wsk - n` → przesuń o `n` elementów do tyłu
- `wsk++` lub `++wsk` → przesuń o 1 element do przodu
- `wsk--` lub `--wsk` → przesuń o 1 element do tyłu
- `wsk1 - wsk2` → różnica w liczbie elementów (typ `ptrdiff_t`)
- Porównania `<`, `>`, `<=`, `>=`, `==`, `!=` między wskaźnikami do tej samej tablicy

Arytmetyka wskaźników działa **tylko w obrębie tej samej tablicy** (lub jeden element za jej końcem). Wyjście poza ten zakres to undefined behaviour.

### Kod

```c
#include <stdio.h>    /* printf */
#include <stddef.h>   /* ptrdiff_t */

int main(void) {
    /* === Przykład 1: tablica jednowymiarowa === */
    float zakupy[6] = {3.12f, 2.50f, 10.0f, 3.22f, 75.12f, 1.00f};

    float *wsk = zakupy;         /* wsk wskazuje na zakupy[0] = 3.12 */

    printf("=== Arytmetyka na tablicy float ===\n");
    printf("*wsk      = %.2f  (zakupy[0])\n", *wsk);

    wsk++;                       /* wsk teraz wskazuje na zakupy[1] */
    printf("Po wsk++: *wsk = %.2f  (zakupy[1])\n", *wsk);

    float trzeci = *++wsk;       /* ++wsk: wsk -> zakupy[2], potem odczyt */
    printf("*++wsk = %.2f  (zakupy[2])\n", trzeci);

    /* wsk wskazuje teraz na zakupy[2]; wsk+2 = zakupy[4] */
    float piaty = *(wsk + 2);    /* nie przesuwa wsk, tylko liczy adres */
    printf("*(wsk+2) = %.2f  (zakupy[4])\n", piaty);

    /* === Przykład 2: tablice wielowymiarowe i arytmetyka === */
    printf("\n=== Tablice wielowymiarowe ===\n");
    int tab1D[10] = {7, 5, 2, 1, 8, 3, 2, 1, 2, 0};
    int *w1 = tab1D;
    w1++;                        /* przesuwa o sizeof(int) = 4 bajty */
    w1++;                        /* przesuwa o kolejne 4 bajty */
    printf("Po dwoch w1++: *w1 = %d  (tab1D[2])\n", *w1);

    /* Wskaźnik na wiersz tablicy 2D — przesuwa o cały wiersz */
    int tab2D[3][5] = {
        {7, 5, 2, 1, 8},         /* wiersz 0 */
        {3, 2, 1, 2, 0},         /* wiersz 1 */
        {9, 4, 6, 3, 7}          /* wiersz 2 */
    };

    int (*wRow)[5] = tab2D;      /* wskaźnik na wiersz 5-elementowy */
    printf("\nPrzed wRow++: (*wRow)[0] = %d  (tab2D[0][0])\n", (*wRow)[0]);
    wRow++;                      /* przesuwa o sizeof(int[5]) = 20 bajtów */
    printf("Po wRow++:    (*wRow)[0] = %d  (tab2D[1][0])\n", (*wRow)[0]);
    wRow++;                      /* teraz tab2D[2] */
    printf("Po wRow++:    (*wRow)[0] = %d  (tab2D[2][0])\n", (*wRow)[0]);

    /* === Przykład 3: różnica wskaźników === */
    printf("\n=== Roznica wskaznikow ===\n");
    int dane[8] = {10, 20, 30, 40, 50, 60, 70, 80};
    int *pocz = &dane[1];        /* wskazuje na dane[1] = 20 */
    int *kon  = &dane[6];        /* wskazuje na dane[6] = 70 */
    ptrdiff_t odl = kon - pocz;  /* ile elementów między nimi */
    printf("kon - pocz = %td elementow (indeksy 1..6)\n", odl);

    /* === Przykład 4: iteracja wskaźnikowa zamiast indeksowej === */
    printf("\n=== Iteracja wskaznikowa ===\n");
    int liczby[5] = {100, 200, 300, 400, 500};
    int *koniec = liczby + 5;    /* jeden za ostatnim elementem — nie dereferencjonuj! */

    for (int *p = liczby; p < koniec; p++) {
        printf("  %d ", *p);     /* odczyt przez dereferencję bieżącej pozycji */
    }
    printf("\n");

    return 0;
}
```

### Przykład wyjścia
```

Po wsk++: *wsk = 2.50  (zakupy[1])

=== Tablice wielowymiarowe ===

Po wRow++:    (*wRow)[0] = 3  (tab2D[1][0])
Po wRow++:    (*wRow)[0] = 9  (tab2D[2][0])

=== Roznica wskaznikow ===
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| Arytmetyka między wskaźnikami do różnych tablic | UB — wynik jest niezdefiniowany | Odejmuj tylko wskaźniki do tej samej tablicy |
| `wsk + n` gdzie n wychodzi poza tablicę | UB (dopuszczalny tylko jeden element za końcem) | Sprawdzaj warunki granic |
| Arytmetyka na `void *` | Kompilator nie zna rozmiaru kroku | Rzutuj na konkretny typ przed arytmetyką |

## 9. Tablica wskaźników vs. wskaźnik do tablicy

### Teoria
To jeden z najbardziej mylących aspektów C — dwie podobnie wyglądające deklaracje mają zupełnie inne znaczenie:

```c
int *twsk[100];      // tablica 100 wskaźników na int
int (*wsk2)[100];    // wskaźnik na tablicę 100 int-ów
```

**Tablica wskaźników** `int *twsk[100]`:
- Mamy 100 oddzielnych wskaźników
- Każdy może wskazywać na zupełnie inne miejsce w pamięci
- Przydatna do tablic stringów, list elementów etc.

**Wskaźnik do tablicy** `int (*wsk2)[100]`:
- Mamy jeden wskaźnik
- Wskazuje na całą tablicę 100 intów
- Przydatny przy wielowymiarowych tablicach (argument funkcji)

**Jak zapamiętać?** Nawiasy mają wyższy priorytet. `(*wsk2)` — najpierw dereferencja (wsk2 jest wskaźnikiem), potem `[100]` — na co wskazuje (tablicę 100 elementów).

### Kod

```c
#include <stdio.h>    /* printf */

int main(void) {
    int x = 10, y = 20, z = 30;
    int tablica[5] = {100, 200, 300, 400, 500};

    /* === TABLICA WSKAŹNIKÓW: int *twsk[N] === */
    /* twsk to tablica pięciu wskaźników — każdy może wskazywać gdzie indziej */
    int *twsk[5];                /* deklaracja tablicy 5 wskaźników na int */

    twsk[0] = tablica;           /* twsk[0] wskazuje na tablica[0] */
    twsk[1] = &x;                /* twsk[1] wskazuje na zmienną x */
    twsk[2] = tablica + 2;       /* twsk[2] wskazuje na tablica[2] (= 300) */
    twsk[3] = &(tablica[3]);     /* twsk[3] wskazuje na tablica[3] (= 400) */
    twsk[4] = &y;                /* twsk[4] wskazuje na zmienną y */

    printf("=== Tablica wskaznikow (int *twsk[5]) ===\n");
    for (int i = 0; i < 5; i++) {
        printf("  *twsk[%d] = %d  (adres: %p)\n", i, *twsk[i], (void *)twsk[i]);
    }

    /* Modyfikacja przez tablicę wskaźników */
    *twsk[1] = 999;              /* zmienia wartość x na 999 */
    printf("\nPo *twsk[1]=999: x = %d\n", x);

    /* === WSKAŹNIK DO TABLICY: int (*wsk2)[N] === */
    /* wsk2 to jeden wskaźnik na tablicę 5 elementów */
    int (*wsk2)[5];              /* deklaracja wskaźnika na tablicę 5 intów */

    wsk2 = &tablica;             /* wsk2 wskazuje na całą tablicę */

    printf("\n=== Wskaznik do tablicy (int (*wsk2)[5]) ===\n");
    printf("Adres tablicy:  %p\n", (void *)&tablica);
    printf("Adres wsk2:     %p\n", (void *)wsk2);

    /* dostęp do elementów: najpierw dereferencja wsk2 (daje tablicę), potem indeks */
    for (int i = 0; i < 5; i++) {
        printf("  (*wsk2)[%d] = %d\n", i, (*wsk2)[i]);
    }

    /* Arytmetyka: wsk2+1 przeskakuje o sizeof(int[5]) = 20 bajtów */
    printf("\nwsk2    = %p\n", (void *)wsk2);
    printf("wsk2+1  = %p  (+%zu bajtow)\n", (void *)(wsk2 + 1),
           (size_t)((char *)(wsk2 + 1) - (char *)wsk2));

    /* Porównanie: w+1 to tylko +4 bajty (jeden int) */
    int *w = tablica;
    printf("w (int*) = %p\n", (void *)w);
    printf("w+1      = %p  (+%zu bajtow)\n", (void *)(w + 1),
           (size_t)((char *)(w + 1) - (char *)w));

    return 0;
}
```

### Przykład wyjścia

|------|-------------|------------|
| `int *wsk2[5]` zamiast `int (*wsk2)[5]` | Bez nawiasów: tablica wskaźników, nie wskaźnik do tablicy | Pamiętaj o nawiasach `(*wsk2)` |
| `wsk2 = tablica` zamiast `wsk2 = &tablica` | Typy niezgodne: `int *` vs `int (*)[5]` | Użyj `&tablica` dla wskaźnika na całą tablicę |
| Dostęp `*wsk2[i]` zamiast `(*wsk2)[i]` | Błędna kolejność operatorów | Najpierw dereferencja wskaźnika: `(*wsk2)[i]` |

## 10. Wskaźniki a tablice wielowymiarowe

### Teoria
Tablica wielowymiarowa w C jest przechowywana **wierszami** (row-major order) jako ciągły blok pamięci. Tablica `int tab[3][5]` jest w pamięci identyczna z `int tab_1d[15]`.

Wskaźnik do wierszy tablicy 2D:
```c
int tab[3][5] = { ... };
int (*w)[5] = tab;   // w wskazuje na tab[0] — wiersz 5-elementowy
```
`w++` przesuwa o `sizeof(int[5]) = 20` bajtów — przeskakuje do następnego wiersza.

### Kod

```c
#include <stdio.h>    /* printf */

int main(void) {
    /* tablica 3 wiersze × 5 kolumn */
    int tab[3][5] = {
        {7, 5, 2, 1, 8},         /* wiersz 0 */
        {3, 2, 1, 2, 9},         /* wiersz 1 */
        {0, 0, 0, 0, 0}          /* wiersz 2 — same zera */
    };

    /* wskaźnik na wiersz 5-elementowy */
    int (*w)[5] = tab;           /* w == &tab[0] — wskazuje na wiersz 0 */

    printf("=== Iteracja wierszami przez wskaznik na wiersz ===\n");
    printf("Startujemy: w wskazuje na tab[0]\n");
    printf("tab[0][0] = (*w)[0] = %d\n", (*w)[0]);

    w++;                         /* przesuwa o 20 bajtów — w teraz = &tab[1] */
    printf("\nPo w++: w wskazuje na tab[1]\n");
    printf("tab[1][0] = (*w)[0] = %d\n", (*w)[0]);

    w++;                         /* przesuwa o kolejne 20 bajtów — w = &tab[2] */
    printf("\nPo w++: w wskazuje na tab[2]\n");
    printf("tab[2][0] = (*w)[0] = %d\n", (*w)[0]);

    /* === Pełna iteracja przez wszystkie elementy tablicy 2D === */
    printf("\n=== Wszystkie elementy tab[3][5] przez wskaznik wierszowy ===\n");
    for (int (*row)[5] = tab; row < tab + 3; row++) {    /* iteracja po wierszach */
        for (int col = 0; col < 5; col++) {              /* iteracja po kolumnach */
            printf("%3d ", (*row)[col]);                 /* dostęp do elementu */
        }
        printf("\n");
    }

    /* === Alternatywnie: zwykły wskaźnik na int (tablica 2D jako 1D) === */
    printf("\n=== Tablica 2D jako ciagly blok pamieci (przez int *) ===\n");
    int *flat = &tab[0][0];      /* wskaźnik na pierwszy element — traktujemy jako 1D */
    for (int i = 0; i < 15; i++) {
        printf("%3d ", flat[i]); /* sekwencyjny dostęp do wszystkich 15 elementów */
        if ((i + 1) % 5 == 0) printf("\n");  /* nowa linia co 5 elementów */
    }

    /* === Pokazanie adresów wierszy === */
    printf("\n=== Adresy wierszy i rozmiary ===\n");
    printf("sizeof(tab)       = %zu bajtow\n", sizeof(tab));        /* 3*5*4 = 60 */
    printf("sizeof(tab[0])    = %zu bajtow\n", sizeof(tab[0]));     /* 5*4 = 20 */
    printf("sizeof(tab[0][0]) = %zu bajtow\n", sizeof(tab[0][0])); /* 4 */
    for (int i = 0; i < 3; i++) {
        printf("Adres tab[%d]: %p\n", i, (void *)tab[i]);
    }

    return 0;
}
```


### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `int *w = tab` dla tablicy 2D | Typ niezgodny: `tab` to `int (*)[5]`, nie `int *` | Użyj `int (*w)[5] = tab;` lub `int *w = &tab[0][0];` |
| Pominięcie wymiaru w typie wskaźnika | Kompilator nie wie, ile bajtów na krok | Zawsze podaj rozmiar kolumn: `int (*w)[COLS]` |
| `(*row)[col]` vs `*row[col]` | Bez nawiasów: `*(row[col])` — indeksowanie tablicy wskaźników | Używaj nawiasów: `(*row)[col]` |

## 11. Wskaźnik do pola struktury — operator `->`

### Teoria
Gdy mamy wskaźnik na strukturę, dostęp do jej pól można uzyskać dwoma równoważnymi sposobami:

```c
struct t_rek *wsk = &dana;

(*wsk).ilosc    // dereferencja wskaźnika, potem dostęp do pola
wsk->ilosc      // operator strzałki — to samo, czytelniejsza forma
```

Operator `->` (strzałka) łączy w sobie dereferencję i dostęp do pola. Jest standardową, preferowaną formą gdy pracujemy ze wskaźnikami na struktury.

### Kod

```c
#include <stdio.h>    /* printf */
#include <string.h>   /* strcpy */

/* definicja struktury z kilkoma polami różnych typów */
struct t_rek {
    int   ilosc;                 /* pole całkowite */
    float masa;                  /* pole zmiennoprzecinkowe */
    char  nazwa[20];             /* pole tekstowe */
};

/* funkcja przyjmująca wskaźnik na strukturę i modyfikująca jej pola */
void ustaw_rekord(struct t_rek *wsk, int il, float m, const char *naz) {
    wsk->ilosc = il;             /* dostęp przez -> zamiast (*wsk).ilosc */
    wsk->masa  = m;
    strcpy(wsk->nazwa, naz);     /* kopiowanie łańcucha do pola */
}

/* funkcja tylko odczytująca — const zabrania modyfikacji */
void wypisz_rekord(const struct t_rek *wsk) {
    printf("Nazwa: %-15s  Ilosc: %3d  Masa: %.2f kg\n",
           wsk->nazwa, wsk->ilosc, wsk->masa);
}

int main(void) {
    struct t_rek dana = {5, 2.54f, "Cukier"};   /* inicjalizacja struktury */
    struct t_rek *wsk = &dana;                   /* wskaźnik na strukturę */

    printf("=== Dwa sposoby dostepu do pola struktury ===\n");

    /* Sposób 1: operator kropki na zderefencjonowanym wskaźniku */
    int zamowienie1 = (*wsk).ilosc;              /* nawiasy konieczne! */
    printf("(*wsk).ilosc  = %d\n", zamowienie1);

    /* Sposób 2: operator strzałki — równoważny, ale czytelniejszy */
    int zamowienie2 = wsk->ilosc;
    printf("wsk->ilosc    = %d\n", zamowienie2);

    float nosnosc = wsk->masa;
    printf("wsk->masa     = %.2f\n", nosnosc);
    printf("wsk->nazwa    = %s\n", wsk->nazwa);

    /* Modyfikacja przez wskaźnik */
    printf("\n=== Modyfikacja przez wskaznik ===\n");
    printf("Przed: ");
    wypisz_rekord(wsk);

    wsk->ilosc += 10;            /* zwiększenie ilości */
    wsk->masa  *= 2.0f;          /* podwojenie masy */
    strcpy(wsk->nazwa, "Sol");   /* zmiana nazwy */

    printf("Po:    ");
    wypisz_rekord(wsk);

    /* Użycie funkcji z wskaźnikiem */
    printf("\n=== Uzycie funkcji ze wskaznikiem ===\n");
    struct t_rek nowy;           /* niezainicjowana struktura */
    ustaw_rekord(&nowy, 100, 50.0f, "Maka");
    printf("Nowy:  ");
    wypisz_rekord(&nowy);

    /* Tablica struktur i wskaźnik do jej elementów */
    printf("\n=== Tablica struktur ===\n");
    struct t_rek magazyn[3];
    ustaw_rekord(&magazyn[0], 200, 100.0f, "Ryż");
    ustaw_rekord(&magazyn[1], 50,  25.5f,  "Olej");
    ustaw_rekord(&magazyn[2], 300, 150.0f, "Makarony");

    for (struct t_rek *p = magazyn; p < magazyn + 3; p++) {
        wypisz_rekord(p);        /* p wskazuje kolejno na każdy element tablicy */
    }

    return 0;
}
```

### Przykład wyjścia
```
=== Dwa sposoby dostepu do pola struktury ===

wsk->ilosc    = 5
wsk->masa     = 2.54
wsk->nazwa    = Cukier

=== Modyfikacja przez wskaznik ===
Przed: Nazwa: Cukier           Ilosc:   5  Masa: 2.54 kg
Po:    Nazwa: Sol              Ilosc:  15  Masa: 5.08 kg

=== Uzycie funkcji ze wskaznikiem ===
Nowy:  Nazwa: Maka             Ilosc: 100  Masa: 50.00 kg

=== Tablica struktur ===
Nazwa: Ryż              Ilosc: 200  Masa: 100.00 kg
Nazwa: Olej             Ilosc:  50  Masa: 25.50 kg
Nazwa: Makarony         Ilosc: 300  Masa: 150.00 kg
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `*wsk.pole` zamiast `(*wsk).pole` | Bez nawiasów: `*(wsk.pole)` — dostęp do pola wskaźnika (błąd) | Używaj `->` lub pamiętaj o nawiasach: `(*wsk).pole` |
| `dana->pole` zamiast `dana.pole` | `->` tylko dla wskaźników, `.` dla wartości | Dla zmiennej strukturalnej: `.`, dla wskaźnika: `->` |
| `strcpy` bez sprawdzenia długości | Przepełnienie bufora | Używaj `strncpy` lub sprawdzaj `strlen` |

## 12. Przekazywanie przez wartość vs. przez wskaźnik

### Teoria
**Przez wartość** (`int suma(int a, int b)`):
- Funkcja otrzymuje **kopie** argumentów
- Oryginalne zmienne w wywołującym nie są zmieniane
- Zmienne lokalne `a`, `b` są tworzone na stosie i niszczone po powrocie funkcji

**Przez wskaźnik** (`int suma_plus(int *a, int *b)`):
- Funkcja otrzymuje **adresy** oryginalnych zmiennych
- Przez dereferencję może modyfikować oryginalne zmienne
- Sam wskaźnik (adres) jest przekazywany przez wartość — ale wskazuje na oryginał

To jest mechanizm umożliwiający funkcjom „zwracanie" wielu wartości i modyfikację zmiennych z innego zakresu.

### Kod

```c
#include <stdio.h>    /* printf */

/* ===== Przekazywanie przez wartość ===== */
/* a i b to lokalne kopie — zmiany nie wpływają na oryginały */
int suma_wartosc(int a, int b) {
    printf("  [w funkcji] a=%d, b=%d (lokalne kopie)\n", a, b);
    a++;                         /* modyfikuje LOKALNĄ kopię, nie oryginalną zmienną */
    b--;
    printf("  [w funkcji po a++, b--] a=%d, b=%d\n", a, b);
    return a + b;                /* wynik oparty na zmodyfikowanych kopiach */
}

/* ===== Przekazywanie przez wskaźnik ===== */
/* a i b to wskaźniki na oryginalne zmienne — zmiany są widoczne na zewnątrz */
int suma_wskaznik(int *a, int *b) {
    printf("  [w funkcji] *a=%d, *b=%d (oryginalne zmienne)\n", *a, *b);
    (*a)++;                      /* modyfikuje ORYGINALNĄ zmienną! */
    (*b)--;
    printf("  [w funkcji po (*a)++, (*b)--] *a=%d, *b=%d\n", *a, *b);
    return *a + *b;
}

/* ===== Przykład: zamiana (swap) — możliwa tylko przez wskaźnik ===== */
void zamien_wartosc(int a, int b) {
    int tmp = a;
    a = b;
    b = tmp;
    /* zmieniliśmy kopie — oryginały bez zmian! */
}

void zamien_wskaznik(int *a, int *b) {
    int tmp = *a;                /* zapisz wartość spod wskaźnika a */
    *a = *b;                     /* wpisz pod adres a to co jest pod adresem b */
    *b = tmp;                    /* wpisz pod adres b zapamiętaną wartość */
}

int main(void) {
    int x = 3, y = 12;
    int wynik;

    /* --- Przez wartość --- */
    printf("=== Przekazywanie przez wartosc ===\n");
    printf("Przed: x=%d, y=%d\n", x, y);
    wynik = suma_wartosc(x, y);
    printf("Po wywolaniu: x=%d, y=%d  (bez zmian!)\n", x, y);
    printf("Wynik: %d\n\n", wynik);

    /* --- Przez wskaźnik --- */
    printf("=== Przekazywanie przez wskaznik ===\n");
    printf("Przed: x=%d, y=%d\n", x, y);
    wynik = suma_wskaznik(&x, &y);  /* przekazujemy ADRESY x i y */
    printf("Po wywolaniu: x=%d, y=%d  (ZMIENIONE!)\n", x, y);
    printf("Wynik: %d\n\n", wynik);

    /* --- Swap przez wartość — nie działa --- */
    int p = 100, q = 200;
    printf("=== Swap przez wartosc (nie dziala) ===\n");
    printf("Przed: p=%d, q=%d\n", p, q);
    zamien_wartosc(p, q);
    printf("Po zamien_wartosc: p=%d, q=%d  (bez zmian!)\n\n", p, q);

    /* --- Swap przez wskaźnik — działa --- */
    printf("=== Swap przez wskaznik (dziala) ===\n");
    printf("Przed: p=%d, q=%d\n", p, q);
    zamien_wskaznik(&p, &q);
    printf("Po zamien_wskaznik: p=%d, q=%d  (zamienione!)\n", p, q);

    return 0;
}
```

### Przykład wyjścia
```
=== Przekazywanie przez wartosc ===
Przed: x=3, y=12
  [w funkcji] a=3, b=12 (lokalne kopie)
  [w funkcji po a++, b--] a=4, b=11
Po wywolaniu: x=3, y=12  (bez zmian!)
Wynik: 15

=== Przekazywanie przez wskaznik ===
Przed: x=3, y=12
  [w funkcji] *a=3, *b=12 (oryginalne zmienne)
  [w funkcji po (*a)++, (*b)--] *a=4, *b=11
Po wywolaniu: x=4, y=11  (ZMIENIONE!)
Wynik: 15

=== Swap przez wartosc (nie dziala) ===
Przed: p=100, q=200
Po zamien_wartosc: p=100, q=200  (bez zmian!)

=== Swap przez wskaznik (dziala) ===
Przed: p=100, q=200
Po zamien_wskaznik: p=200, q=100  (zamienione!)
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| Próba swapa przez wartość | Kopie są zamieniane, oryginały nie | Zawsze używaj wskaźników do modyfikacji zewnętrznych zmiennych |
| `suma_wskaznik(x, y)` zamiast `suma_wskaznik(&x, &y)` | Przekazujesz wartość jako adres — UB lub błąd kompilacji | Gdy funkcja przyjmuje `int *`, przekazuj `&zmienna` |
| `*a++` zamiast `(*a)++` w funkcji | Przesuwa wskaźnik, nie inkrementuje wartości | Zawsze nawiasy: `(*a)++` |

## 13. Tablica jako argument funkcji

### Teoria
Gdy przekazujemy tablicę do funkcji, automatycznie **zamienia się ona na wskaźnik** do pierwszego elementu (array decay). Oznacza to:
- Funkcja **nie zna rozmiaru** tablicy — musimy go przekazać osobnym parametrem
- Modyfikacje w funkcji **zmieniają oryginalną tablicę** (bo wskaźnik wskazuje na oryginał)
- `sizeof(tab)` wewnątrz funkcji da rozmiar wskaźnika (8), nie tablicy!

Deklaracje `void f(int tab[], int n)` i `void f(int *tab, int n)` są **dokładnie równoważne** — kompilator traktuje je tak samo.

### Kod

```c
#include <stdio.h>    /* printf */

/* === Funkcja z tablicą jako argumentem — notacja tablicowa === */
/* int t[] w parametrze jest tożsame z int *t — kompilator zamieni to samo */
double suma_tablicy(float t[], int dl, int start, int stop) {
    double s = 0.0;
    /* zabezpieczenie granic: stop nie może przekraczać długości */
    stop  = (stop < dl)    ? ((stop < 0)   ? 0    : stop)  : dl - 1;
    /* start nie może być większy od stop */
    start = (start <= stop) ? ((start < 0) ? 0    : start) : stop;

    for (; start <= stop; start++) {
        s += t[start];           /* dostęp przez indeks — czytelny sposób */
    }
    return s;
}

/* === Ta sama funkcja — notacja wskaźnikowa === */
double suma_wskaznikiem(float *t, int dl, int start, int stop) {
    double s = 0.0;
    stop  = (stop < dl)    ? ((stop < 0)   ? 0    : stop)  : dl - 1;
    start = (start <= stop) ? ((start < 0) ? 0    : start) : stop;

    /* iteracja przez wskaźnik zamiast indeksu */
    for (float *wsk = t + start; wsk <= t + stop; wsk++) {
        s += *wsk;               /* dereferencja bieżącej pozycji wskaźnika */
    }
    return s;
}

/* === Funkcja wypisująca tablicę === */
void wypisz_tablice(float *t, int n) {
    printf("[");
    for (int i = 0; i < n; i++) {
        printf("%.2f%s", t[i], (i < n - 1) ? ", " : "");
    }
    printf("]\n");
}

/* === Funkcja modyfikująca tablicę (demonstracja, że zmiany są widoczne) === */
void skaluj_tablice(float *t, int n, float wspolczynnik) {
    for (int i = 0; i < n; i++) {
        t[i] *= wspolczynnik;    /* modyfikacja oryginału — t wskazuje na zewnętrzną tablicę */
    }
}

int main(void) {
    float zakupy[6] = {3.12f, 2.50f, 10.0f, 3.22f, 75.12f, 1.00f};
    int   n = 6;                 /* rozmiar tablicy — musimy przekazywać ręcznie */

    printf("Tablica zakupy: ");
    wypisz_tablice(zakupy, n);

    /* Suma wszystkich elementów (indeksy 0..5) */
    double s1 = suma_tablicy(zakupy, n, 0, 5);
    printf("\nSuma [0..5] (indeksy)   = %.2f\n", s1);

    double s2 = suma_wskaznikiem(zakupy, n, 0, 5);
    printf("Suma [0..5] (wskazniki) = %.2f\n", s2);

    /* Suma środkowych elementów */
    double s3 = suma_tablicy(zakupy, n, 1, 4);
    printf("Suma [1..4]             = %.2f\n", s3);

    /* Demonstracja, że funkcja modyfikuje oryginalną tablicę */
    printf("\nPrzed skalowaniem: ");
    wypisz_tablice(zakupy, n);

    skaluj_tablice(zakupy, n, 2.0f);  /* mnożenie przez 2 */

    printf("Po skalowaniu x2:  ");
    wypisz_tablice(zakupy, n);       /* oryginalna tablica jest zmieniona! */

    /* sizeof wewnątrz vs. na zewnątrz funkcji */
    printf("\nsizeof(zakupy) w main:        %zu bajtow\n", sizeof(zakupy)); /* 6*4=24 */
    /* Uwaga: sizeof(t) wewnątrz funkcji da 8 (rozmiar wskaźnika), nie 24 */

    return 0;
}
```

### Przykład wyjścia
```
Tablica zakupy: [3.12, 2.50, 10.00, 3.22, 75.12, 1.00]

Suma [0..5] (indeksy)   = 94.96
Suma [0..5] (wskazniki) = 94.96
Suma [1..4]             = 90.84

Przed skalowaniem: [3.12, 2.50, 10.00, 3.22, 75.12, 1.00]
Po skalowaniu x2:  [6.24, 5.00, 20.00, 6.44, 150.24, 2.00]

sizeof(zakupy) w main:        24 bajtow
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| `sizeof(t)` wewnątrz funkcji dla rozmiaru tablicy | Daje 8 (rozmiar wskaźnika), nie rozmiar tablicy | Przekazuj rozmiar jako osobny parametr `int n` |
| Brak parametru `n` (rozmiar) | Funkcja nie wie, gdzie kończy się tablica | Zawsze przekazuj rozmiar tablicy |
| Modyfikacja tablicy przez funkcję przez nieuwagę | Array decay — tablice są przekazywane przez wskaźnik | Jeśli nie chcesz modyfikować, użyj `const float *t` |

## 14. Dynamiczna alokacja pamięci — `malloc` / `free`

### Teoria
Zmienne lokalne są alokowane na stosie — znikają po wyjściu z funkcji. Czasem potrzebujemy pamięci o **rozmiarze określanym w trakcie działania** programu lub pamięci, która przetrwa dłużej niż jedna funkcja. Do tego służy alokacja na **stercie** (heap).

Kluczowe funkcje z `<stdlib.h>`:
- `void *malloc(size_t rozmiar)` — alokuje `rozmiar` bajtów, zwraca `void *` (NULL jeśli błąd)
- `void *calloc(size_t n, size_t rozmiar)` — jak malloc, ale zeruje pamięć; alokuje `n * rozmiar` bajtów
- `void *realloc(void *ptr, size_t nowy_rozmiar)` — zmienia rozmiar bloku
- `void free(void *ptr)` — zwalnia pamięć

**Złota zasada**: każde `malloc`/`calloc`/`realloc` musi mieć odpowiadające `free`. Brak `free` = **wyciek pamięci** (memory leak).

### Kod

```c
#include <stdio.h>    /* printf */
#include <stdlib.h>   /* malloc, calloc, realloc, free */
#include <string.h>   /* memset */

/* struktura do demonstracji dynamicznej tablicy struktur */
struct t_rek {
    int   ilosc;
    float masa;
};

int main(void) {
    /* === malloc: alokacja bez zerowania === */
    printf("=== malloc ===\n");
    int n = 5;                               /* rozmiar do określenia w trakcie działania */
    int *tablica = (int *) malloc(n * sizeof(int)); /* alokuj n intów */

    if (tablica == NULL) {                   /* ZAWSZE sprawdzaj wynik malloc! */
        printf("Blad alokacji!\n");
        return 1;
    }

    printf("Zaalokowano %d intow (%zu bajtow)\n", n, n * sizeof(int));

    /* wypełnienie i użycie */
    for (int i = 0; i < n; i++) {
        tablica[i] = (i + 1) * 100;         /* 100, 200, 300, 400, 500 */
    }
    printf("Tablica: ");
    for (int i = 0; i < n; i++) printf("%d ", tablica[i]);
    printf("\n");

    free(tablica);                           /* zwolnienie pamięci */
    tablica = NULL;                          /* zerowanie wskaźnika — dobra praktyka */

    /* === calloc: alokacja z zerowaniem === */
    printf("\n=== calloc ===\n");
    float *flt = (float *) calloc(4, sizeof(float)); /* 4 floaty, wyzerowane */

    if (flt == NULL) {
        printf("Blad alokacji!\n");
        return 1;
    }

    printf("Zaalokowano i wyzerowano 4 floaty\n");
    for (int i = 0; i < 4; i++) printf("%.1f ", flt[i]); /* wszystkie 0.0 */
    printf("\n");

    flt[0] = 1.5f; flt[1] = 2.5f; flt[2] = 3.5f; flt[3] = 4.5f;
    for (int i = 0; i < 4; i++) printf("%.1f ", flt[i]);
    printf("\n");

    free(flt);
    flt = NULL;

    /* === realloc: zmiana rozmiaru === */
    printf("\n=== realloc ===\n");
    int *dane = (int *) malloc(3 * sizeof(int));
    if (dane == NULL) { return 1; }

    dane[0] = 10; dane[1] = 20; dane[2] = 30;
    printf("Przed realloc (3 elementy): ");
    for (int i = 0; i < 3; i++) printf("%d ", dane[i]);
    printf("\n");

    /* rozszerzenie tablicy do 6 elementów */
    int *tmp = (int *) realloc(dane, 6 * sizeof(int));
    if (tmp == NULL) {
        free(dane);              /* jeśli realloc zawiedzie, stary blok nadal istnieje */
        return 1;
    }
    dane = tmp;                  /* używaj tmp, bo realloc może zwrócić nowy adres */

    dane[3] = 40; dane[4] = 50; dane[5] = 60;
    printf("Po realloc (6 elementow): ");
    for (int i = 0; i < 6; i++) printf("%d ", dane[i]);
    printf("\n");

    free(dane);
    dane = NULL;

    /* === Dynamiczna tablica struktur === */
    printf("\n=== Dynamiczna tablica struktur ===\n");
    int ile = 3;
    struct t_rek *wsk = (struct t_rek *) malloc(ile * sizeof(struct t_rek));
    if (wsk == NULL) { return 1; }

    /* wypełnienie przez wskaźnik z operatorem strzałki */
    for (int i = 0; i < ile; i++) {
        wsk[i].ilosc = (i + 1) * 10;        /* 10, 20, 30 */
        wsk[i].masa  = (i + 1) * 2.5f;      /* 2.5, 5.0, 7.5 */
    }

    printf("Dynamiczna tablica %d rekordow:\n", ile);
    for (int i = 0; i < ile; i++) {
        printf("  [%d]: ilosc=%d, masa=%.1f\n", i, wsk[i].ilosc, wsk[i].masa);
    }

    free(wsk);
    wsk = NULL;

    printf("\nWszystka pamiec zwolniona.\n");
    return 0;
}
```

```

=== calloc ===

=== realloc ===
```

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| Brak sprawdzenia `NULL` po malloc | Dereferencja NULL → crash | Zawsze `if (ptr == NULL) { return ...; }` |
| `malloc(n)` zamiast `malloc(n * sizeof(typ))` | Za mała alokacja → przepełnienie bufora | Zawsze używaj `sizeof`: `malloc(n * sizeof(int))` |
| Brak `free` | Wyciek pamięci | Każdy `malloc` musi mieć `free` |
| `free` dwa razy (double free) | Undefined behaviour — crash | Zeruj po free: `ptr = NULL;` |
| `dane = realloc(dane, ...)` | Jeśli realloc zawiedzie, tracisz oryginalny wskaźnik | Używaj tymczasowego: `tmp = realloc(dane, ...); if(tmp) dane=tmp;` |

## 15. Zarządzanie pamięcią — stos i sterta

### Teoria
Pamięć programu C jest podzielona na kilka obszarów:

| Obszar | Nazwa ang. | Zawiera | Zarządzanie |
|--------|-----------|---------|------------|
| **Stos** | Stack | Zmienne lokalne, argumenty funkcji, adresy powrotu | Automatyczne (LIFO) |
| **Sterta** | Heap | Pamięć dynamiczna (`malloc`/`calloc`) | Ręczne (`free`) |
| **Segment danych statycznych** | Static/BSS | Zmienne globalne i `static` | Przez cały czas życia programu |
| **Segment kodu** | Text/Code | Instrukcje programu, stałe `const` | Tylko do odczytu |

**Stos** (Stack):
- Działa jak LIFO (Last In, First Out) — ostatni wchodzi, pierwszy wychodzi
- Każde wywołanie funkcji tworzy **ramkę stosu** (stack frame) z jej lokalnymi zmiennymi
- Po powrocie funkcji ramka jest automatycznie niszczona
- Rozmiar ograniczony (typowo 1–8 MB) — głęboka rekurencja → stack overflow

**Sterta** (Heap):
- Jawna alokacja przez `malloc`/`calloc`/`realloc`
- Jawna dealokacja przez `free`
- Może rosnąć dynamicznie (ograniczona dostępną pamięcią RAM)
- Podatna na **fragmentację** — wolne bloki rozrzucone po pamięci

### Kod

```c
#include <stdio.h>    /* printf */
#include <stdlib.h>   /* malloc, free */

/* Zmienna globalna — pamięć statyczna, istnieje przez cały czas działania programu */
int zmienna_globalna = 42;

/* Stała — w segmencie kodu (read-only) lub w segmencie danych */
const int STALA = 100;

/* Funkcja demonstrująca zmienne na stosie */
void demonstracja_stosu(int poziom) {
    int zmienna_lokalna = poziom * 10;   /* na stosie — niszczona po powrocie */
    static int licznik = 0;             /* w pamięci statycznej — przetrwa wywołania */
    licznik++;

    printf("Poziom %d: zmienna_lokalna=%d, licznik=%d\n",
           poziom, zmienna_lokalna, licznik);
    printf("  adres zmienna_lokalna: %p (stos)\n", (void *)&zmienna_lokalna);
    printf("  adres licznik:         %p (statyczna)\n", (void *)&licznik);

    if (poziom < 3) {
        demonstracja_stosu(poziom + 1);  /* rekurencja — każde wywołanie: nowa ramka */
    }
    /* Po powrocie z rekurencji: zmienna_lokalna jest niszczona */
}

int main(void) {
    printf("=== Segmenty pamieci ===\n\n");

    /* Zmienna lokalna w main — na stosie */
    int lokalna_main = 7;

    /* Dynamiczna — na stercie */
    int *sterta = (int *) malloc(3 * sizeof(int));
    sterta[0] = 100; sterta[1] = 200; sterta[2] = 300;

    printf("Adres zmiennej globalnej:    %p (segment statyczny)\n",
           (void *)&zmienna_globalna);
    printf("Adres stalej STALA:          %p (segment kodu/danych)\n",
           (void *)&STALA);
    printf("Adres zmiennej lokalnej:     %p (stos)\n",
           (void *)&lokalna_main);
    printf("Adres tablicy na stercie:    %p (sterta)\n",
           (void *)sterta);

    /* Typowy układ: kod i statyczne < sterta < ... < stos (od wysokich adresów) */
    /* Dokładny układ zależy od systemu operacyjnego i kompilatora */

    printf("\n=== Demonstracja stosu (ramki rekurencyjne) ===\n");
    demonstracja_stosu(1);

    printf("\n=== Niebezpieczny wskaznik do zmiennej lokalnej ===\n");
    /* UWAGA: poniższy kod jest celowo błędny — pokazuje co NIE robić */
    /* Zwracanie adresu zmiennej lokalnej jest undefined behaviour! */
    /* Zmienna lokalna przestaje istnieć po powrocie z funkcji */
    /* W prawdziwym kodzie kompilator ostrzeże: warning: address of local */

    printf("Sterta istnieje dopoki nie wywolamy free:\n");
    printf("sterta[0]=%d, sterta[1]=%d, sterta[2]=%d\n",
           sterta[0], sterta[1], sterta[2]);

    free(sterta);                        /* zwolnienie pamięci sterty */
    sterta = NULL;
    printf("Po free: sterta = NULL\n");

    printf("\n=== Roznica stosu i sterty ===\n");
    printf("%-30s %-15s %s\n", "Cecha", "Stos", "Sterta");
    printf("%-30s %-15s %s\n", "Zarzadzanie", "automatyczne", "reczne");
    printf("%-30s %-15s %s\n", "Rozmiar", "ograniczony", "do RAM");
    printf("%-30s %-15s %s\n", "Fragmentacja", "nie", "tak");
    printf("%-30s %-15s %s\n", "Szybkosc allokacji", "szybka", "wolniejsza");
    printf("%-30s %-15s %s\n", "Czas zycia", "do konca bloku", "do free()");

    return 0;
}
```

Zauważ, że adres `licznik` (static) jest taki sam we wszystkich wywołaniach — istnieje jedna kopia w pamięci statycznej. Adresy `zmienna_lokalna` maleją z poziomem — stos rośnie w dół (na x86).

### Typowe błędy
| Błąd | Dlaczego zły | Jak unikać |
|------|-------------|------------|
| Zwracanie adresu zmiennej lokalnej | Zmienna niszczona po powrocie — dangling pointer | Zwracaj dane przez `malloc` (sterta) lub przez parametr-wskaźnik |
| Stack overflow przez głęboką rekurencję | Stos ma ograniczony rozmiar | Używaj iteracji lub zwiększ rozmiar stosu |
| Wyciek pamięci (brak `free`) | Sterta się zapełnia — program może crash-ować | Każde `malloc` = jedno `free` |
| Dostęp po `free` (use-after-free) | UB — możliwy crash lub błąd bezpieczeństwa | Zeruj wskaźnik po `free`: `ptr = NULL` |

## Podsumowanie — ściągawka operatorów

```
&zmienna     → adres zmiennej (zwraca wskaźnik)
*wskaznik    → wartość pod wskaźnikiem (dereferencja)
wskaznik->pole → pole struktury przez wskaźnik (skrót od (*wskaznik).pole)
wskaznik[i]  → element i-ty (skrót od *(wskaznik + i))
wskaznik + n → adres n elementów dalej
wskaznik - n → adres n elementów wcześniej
wsk1 - wsk2  → różnica w liczbie elementów (ptrdiff_t)
```

## Podsumowanie — deklaracje wskaźników

```c
int *p;              // wskaźnik na int
int **pp;            // wskaźnik na wskaźnik na int
int *t[10];          // tablica 10 wskaźników na int
int (*pt)[10];       // wskaźnik na tablicę 10 intów
int (*pf)(int, int); // wskaźnik na funkcję int(int,int)
const int *pc;       // wskaźnik do stałej (wartości nie zmienimy przez wskaźnik)
int * const cp;      // stały wskaźnik (adresu nie zmienimy)
const int * const ccp; // stały wskaźnik do stałej
void *vp;            // wskaźnik bez typu
```
